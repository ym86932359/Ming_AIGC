## 使用规则

1. 新图先用本人照片锁定脸，再用日常总卡补身体、服装与发型。
2. 小丑状态额外加载正式全脸妆造卡，只继承妆面几何和颜色。
3. 表情卡可以改变面部肌肉、视线、泪水和呼吸，不改变脸型、年龄、发型、天然瞳色或妆面结构。
4. 所有新图先标记`待审`；用户确认后才进入关联包的`锁定`版本。
