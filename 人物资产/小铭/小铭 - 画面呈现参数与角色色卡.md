# 小铭 - 画面呈现参数与角色色卡

> 本页只定义“小铭怎样呈现最稳定、最好看”。日常和小丑状态共享身份与天然肤色；切换妆造时，只改变妆层、局部色彩和光比。

> [!info] 当前边界
> 下列v04参数是关联包当前生产光色合同；历史v03参数仅供旧镜头追溯。

## 0. v04当前光色接口

| 层级 | v04当前锁定值 |
| --- | --- |
| 共享画风／角色变体 | `ZERO-ECHO-VIS-NEO-CHINESE-CYBER-GLAZE-02`／`XIAOMING-STYLE-NEO-CHINESE-BARTENDER-01` |
| 角色固有色 | 烟墨黑、漆铁灰、烟青玉、旧金、暗朱砂、月白与服从本人照片的自然肤色；布光不得重定义这些固有色 |
| 专属材质 | [[个人创作/ai铭仔/资产库/材质资产/核心材质/墨曜青玉琉璃]]；`ZERO-ECHO-MAT-OBSIDIAN-JADE-GLAZE-01` |
| 日常身份主光 | 左前35°、俯角20°、4300K大型柔光、100%；优先保护本人脸型、毛孔、肤色、眼白、天然深棕虹膜和唇色 |
| 小丑身份主光 | 正前偏左15°、4100K大型柔光、100%；同时保护真实骨相、象牙妆层、墨黑眼翼、朱砂笑弧和天然眼眸 |
| 辅光 | 日常右前5200K、主辅比4:1；小丑右前5000K、主辅比2.5:1；只抬暗部可读性，不改写肤色或妆色 |

| 品红折射上限 | 只在琉璃斜角折射交界出现，占人物画面低于2%；不得进入脸、眼白、天然虹膜、唇色或大面积织物 |
| 肤色保护 | 日常肤色保持60—65 IRE；小丑象牙妆层保持62—70 IRE；开启肤色保护，禁止青紫、朱砂低位光或场景色污染面部中间调 |
| 琉璃高光上限 | 候选资产中所有墨曜青玉琉璃折射高光合计不超过画面10%，保留烟黑内部、青玉窄边、旧金／黑钛框架与可读倒角，不允许剪切为纯白 |
| 场景光继承 | 人物进入酒馆、雨夜或记忆画面时继承现场方向、色温、遮挡和环境反射；身份主光优先，场景光不得重写角色固有色、妆色、天然眼眸或材质底色 |

当前接口遵循“角色固有色→身份主光→琉璃轮廓光→场景光”的分层记录；场景光只能适配，不能替换前三级。

## 1. 全局继承

- 全局色卡：[[个人创作/ai铭仔/资产库/色彩资产/零点回声酒馆 - 全局色卡池]]。
- 继承：深夜琥珀、烟木黑、雨夜蓝灰、故障青、陈年酒红、暖骨象牙、旧金。
- 禁止：商业青橙、大面积洋红霓虹、荧光绿、亮蓝肤色、纯黑死影。

## 2.1 历史黑曜青玉琉璃材质参数（v03）

- 琉璃基色：内部烟黑`#11161B`，边缘青玉`#43C5C7`，角度偏移品红`#D24B9A`，折射高光不超过画面10%。
- 厚度：领口与肩缘2.5—4毫米，前臂观察窗4—6毫米；粗糙度0.16—0.32，透射率18—42%，折射率1.49—1.56。
- 支撑：旧黄铜或拉丝黑钛细框必须与织物缝合点、磁吸扣和关节方向一致。
- 光效：青紫只沿琉璃边缘、发丝和外轮廓出现；不得进入面部中间调、眼白和天然虹膜。

## 2.2 v04当前墨曜青玉琉璃参数

- 材质ID：`ZERO-ECHO-MAT-OBSIDIAN-JADE-GLAZE-01`；只安装在双层斜襟、半云肩肩缘、前臂窗、腰封嵌片和鞋跟窗。
- 厚度2.5—6毫米、倒角0.8—1.5毫米、粗糙度0.18—0.30、金属度0、基础透射20%—34%、折射率1.50—1.55。
- 内部为烟黑纵深，边缘为低亮青玉；旧金／黑钛框架必须完整包边、夹持或铆接，并产生方向一致的真实投影。
- 门纹、山水等高线、流线与纵向层叠只属于小铭；正、侧、背视图中的位置、厚度、连接点和受光必须连续。

## 4. 日常身份光

| 光源 | 参数 | 保护目标 |
| --- | --- | --- |
| 主光 | 左前35°、俯角20°、4300K、中等柔光、100% | 脸型、眉眼、鼻翼与真实肤色 |
| 辅光 | 右前20°、5000K、主辅比4:1 | 眼窝和黑衣纹理 |
| 轮廓光 | 左后45°、7000K、35% | 黑发分束、肩线、玻璃边缘 |
| 眼神光 | 正前略高的小型柔光点、低强度 | 天然深棕虹膜与角膜湿润 |
| 电路反光 | 右侧低亮烟青玉、8%以内 | 只描出太阳穴、护臂或衣缘 |

### 日常表情卡专用

- 背景：中性烟灰，不出现酒馆场景。
- 光比：`3:1`，略比剧情光平，保证24格眉眼、鼻翼和唇部肌肉可读。
- 曝光：`-0.2EV`；肤色`60–65 IRE`；黑衣最低可读`9 IRE`。
- 饱和度`-8`、自然饱和`+6`、对比`+8`、高光`-14`、阴影`+10`、黑位`-4`。

## 5. 小丑妆身份光

| 光源 | 参数 | 保护目标 |
| --- | --- | --- |
| 主光 | 正前偏左15°、4100K、大型柔光、100% | 全脸妆几何、真实骨相与裂纹 |
| 辅光 | 右前15°、5000K、主辅比2.5:1 | 墨黑眼翼内仍看清眼神 |
| 轮廓光 | 左后45°、7200K、38% | 黑发与暗背景分离 |
| 眼神光 | 双眼上方小矩形柔光、低强度 | 天然虹膜、黑瞳、眼白 |
| 低位朱砂反光 | 正前下方、2200–2600K、0–12% | 只在剧情高潮使用，不用于标准表情卡 |

### 小丑表情卡专用

- 背景：无纹理炭黑到烟灰渐变。
- 所有格使用同一中性身份光，不随情绪改变颜色或方向。
- 曝光：`-0.3EV`；象牙底妆`62–70 IRE`；墨黑眼翼最低`8 IRE`；朱砂不剪切。
- 饱和度`-5`、自然饱和`+8`、对比`+12`、高光`-18`、阴影`+7`、黑位`-6`。

## 6. 最强眼神冲击参数

适用于剧集尾声语录，不作为表情库的标准光：

1. 关掉大部分背景实景灯，环境降低约`1.5档`。
2. 用`6200K`窄条柔光横向掠过双眼，控制在眉骨至鼻梁上段，不照亮整张脸。
3. 正前下方加入`2400K`低位朱砂反光，强度为主光`15–20%`，只抬起鼻尖、唇部笑弧和下颌。
4. 左后方`7200K`轮廓光保持`40–45%`，分离黑发与肩线。
5. 眼神光保持小而清晰；双眼不能变成黑洞或发光眼。
6. 画面整体曝光`-0.7EV`，但眼白、天然虹膜、朱砂与象牙底妆仍保留层次。

强冲击来自“眼区冷、鼻唇低位暖、背景沉黑”的三段对比，不来自提高全画面饱和度。

## 7. 面部材质保护

- 额头、鼻翼、上唇和面颊保留毛孔与妆层颗粒。
- 墨黑眼翼边缘清楚但带真实手绘微差，不做矢量贴纸。
- 朱砂笑弧位于妆层表面，不出现凹陷、出血或皮肤撕裂。
- 象牙底妆的细裂纹只在近看可见，远看仍保持完整脸型。
- 任何光线都不得覆盖天然深棕虹膜、黑瞳和自然眼白。
- 年轻化只减少眼下疲态并提升软组织紧致度；不放大眼睛、不缩短中庭、不改鼻翼和下颌。
- 形状记忆衣料、柔性黑钛与电子织线必须接受同一实景光，不能呈现为游戏CG贴图。

## 8. 禁区

- 用红光或蓝光彻底吞掉五官。
- 将象牙妆拍成纯白平面，将墨黑眼翼压成没有眼神的死黑。
- 让青玉电路发出霓虹灯强光，污染皮肤与眼白。
- 为追求反派感添加绿发、紫色西装、割嘴、血迹或现成小丑符号。
- 两张表情卡使用不同白平衡、不同发色或不同基础领口。
