## 调用规则

1. 下游分镜先读取对应人物模型关联包，不从目录随机选择图片。
2. 小铭妆前、妆后共用同一连续性编号；妆容是临时状态，不是第二个人。
3. 小铭使用两张独立24格表情卡；每张卡只对应一种固定妆造，不能在同一网格内随机换妆。
4. 表情调用统一写为`角色连续性编号＋表情编号＋中文情绪名`，例如：`LINYE-S1-EP01 / E24 删除未来`；编号与中文名必须同时出现，避免只凭编号误调用。
5. 每个角色的表情提示词页保存“表情肌表现＋剧情用途”，生成分镜前必须一并读取。
6. 角色固有色服从[[个人创作/ai铭仔/资产库/色彩资产/零点回声酒馆 - 全局色卡池|全局色卡池]]，场景光服从[[个人创作/ai铭仔/世界观与视觉/零点回声酒馆 - 世界观与环境视觉圣经|世界视觉圣经]]。
7. 分镜必须通过角色关联包读取材质、核心道具和`performance_state_namespace`，不得在镜头提示词中另造服装材质或道具版本。
8. 配饰通过人物配饰资产中心读取；故事道具兼任佩戴物时，人物配饰卡只记录挂接和受力，不复制道具状态机。
9. 人物版本、表情编号、配饰编号和镜头元素必须同时写入逐镜调用表，禁止只凭图片文件名调用。
