# 零点回声酒馆 - 4套场景Look Frame与LUT基准

## 同一测试画面

四套Look必须使用同一张源画面：小铭日常形态中近景、深色服装、真实肤色、胡桃木吧台、烟熏玻璃、旧黄铜、透明记忆杯、白／灰参考卡和少量雨窗。只改变调色、灯光状态和合法VFX，不换人物、材质、构图或曝光基础。

## 对照输出

每套输出四格：`原始中性`、`Look Frame`、`肤色／黑衣／玻璃／黄铜局部`、`禁用对照`。另保存：

- 预演LUT或等效参数，只用于快速预览。
- 正式调色参考，不要求一键套用。
- 无LUT中性母版，防止调色烘焙后无法回退。

## 技术目标

| 项目 | 基准 |
| --- | --- |
| 肤色 | 58—65 IRE；角色肤色差异保留 |
| 黑位 | 4—8 IRE；黑衣织物可读 |
| 高光 | 玻璃／黄铜／妆层≤88 IRE |
| 故障青 | 普通状态≤画面5%，脸部≤2% |
| 饱和 | 不以全局高饱和制造高级感 |
| A/B | 同一源图、同一构图、同一人物，才可比较 |

通过后将四张场景色卡状态从`待审`改为`锁定`，并记录Look Frame、LUT／参数、尺寸、SHA与人工结论。
