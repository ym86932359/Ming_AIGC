# 珠光流动琉璃

> 姜禾v04专属琉璃材质。材质以“开放、变化、保留呼吸”为逻辑，不覆盖姜禾24岁身份、风衣正史或相机负重关系。

## 物理属性

- 采用2.5—5毫米厚度，边缘倒角0.8—1.4毫米；可拆片层必须保留真实截面和连接间隙。
- 表面粗糙度0.24—0.38，金属度0，基础透射32%—46%，折射率1.48—1.53。
- 主体为乳烟半透明，内部有低密度珠光雾层；边缘为柔青灰边缘，不得变成高饱和青色发光线。
- 只允许极少品红折射出现在斜视倒角和环境反射交界，面积不得压过柔青灰与乳烟主体。
- 表面允许细微风蚀、指纹和雨滴痕；禁止婚纱水钻、亮片膜或无瑕塑料雨衣。

## 固定结构与几何

- 所有片层由黑钛柔性框架包边、铰接或插接；框架允许服从风衣动作弯曲，但必须显示固定点与真实受力。
- 几何只用开放弧面、风向线和可拆透明层。
- 片层开口方向必须服从身体动作与来风方向；相机肩带穿过服装承重结构，不得被透明层遮断或悬浮。

## 使用范围

- 用于开放式风衣领边、可拆短披肩、前臂观察窗、衣摆风向片与相机肩带保护片。
- 不覆盖脸、头发、手掌、胸腹大面、关节内侧、整件风衣或相机镜头主体。
- 可拆透明层必须在正、侧、背视图保持数量、连接点和开口方向连续，不得逐视图随机增减。

## 光线响应

| 条件 | 固定响应 |
| --- | --- |
| 2800—3200K暖光 | 乳烟半透明呈暖珠灰，柔青灰边缘保持中性；不变纯白或香槟金 |
| 4100—4300K身份光 | 珠光雾层、柔青灰边缘、黑钛柔性框架和真实厚度同时可辨，不抬亮肤色 |
| 6800—7600K冷轮廓光 | 开放弧面与风向线出现柔青灰窄边，品红折射保持极少且只在倒角交界 |
| 干态 | 乳烟层次、细微风蚀、连接点与开放路径清楚，不像婚纱亮片 |
| 湿态 | 高光增强约9%，透射降低约4%，雨滴沿弧面滑向开口，不把片层变成透明湿衣 |
| 近景 | 显示珠光雾层、真实截面、柔性框架铰接、指纹与雨滴痕 |
| 远景 | 只保留乳烟轻体块、柔青灰开放边和清楚风向，不制造仙气光尘 |

## 禁止项

- 禁止婚纱亮片、仙女感与林野封闭矩形。
- 禁止水晶翅膀、漂浮薄纱、星尘粒子、纯白爆光、法术光环和无固定点透明层。
- 禁止把姜禾的开放弧面、风向线或可拆透明层复制给小铭与林野。
