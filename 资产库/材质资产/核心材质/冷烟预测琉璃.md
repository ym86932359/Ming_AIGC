# 冷烟预测琉璃

> 林野v04专属琉璃材质。材质以“封闭、可预测、可计算”为逻辑，不覆盖林野26岁身份、服装正史或存储环功能。

## 物理属性

- 采用2.5—6毫米厚度，边缘倒角0.6—1.2毫米；窄片与同心层均须显示真实截面。
- 表面粗糙度0.22—0.34，金属度0，基础透射14%—26%，折射率1.49—1.54。
- 主体为烟灰低透射，内部为预测青内层；预测青位于材质内部刻度与同心层，不是表面贴纸或全片自发光。
- 斜切倒角允许品红斜角折射，但品红只占边缘折射的极小部分，不得覆盖烟灰主体或人物皮肤。
- 表面允许精细磨损、指纹和校准划痕；禁止冰糖、塑料板和无瑕科幻屏幕。

## 固定结构与几何

- 所有片层由黑钛闭合框架包边、压合或铰接，框架必须形成清楚的中轴闭合和承重路径。
- 几何只用窄矩形、平行切面与同心层。
- 几何节距、层距与倒角方向在正、侧、背视图保持一致，不得随机生成开放曲线或悬浮数据片。

## 使用范围

- 用于对襟立领闭合片、竹简式层叠片、前臂预测窗、外套中轴窄条与42毫米未来存储环护片。
- 不覆盖脸、头发、手掌、关节内侧、胸腹大面或需要连续柔性弯折的主织物。
- 存储环护片必须服从42毫米尺寸与右手持握，不得扩大成武器、护盾或魔法法器。

## 光线响应

| 条件 | 固定响应 |
| --- | --- |
| 2800—3200K暖光 | 烟灰保持冷中性，预测青降为低亮内层，黑钛框架不变铜色 |
| 4100—4300K身份光 | 烟灰低透射、预测青内层、品红斜角折射和闭合框架均可辨，人物肤色保持自然 |
| 6800—7600K冷轮廓光 | 平行切面与同心层边缘增强；品红只出现在极窄斜角，不形成虹彩大面 |
| 干态 | 刻度、校准划痕、倒角和层距清楚，主体不出现镜面复制 |
| 湿态 | 高光增强约10%，透射降低约5%，水珠沿闭合框架停留且不破坏刻度连续性 |
| 近景 | 显示烟灰内部、预测青层、品红斜角、黑钛压合与真实截面 |
| 远景 | 只保留封闭冷灰体块、预测青窄层与精确中轴，不放大HUD或光效 |

## 禁止项

- 禁止小铭门纹、云肩和姜禾开放弧面。
- 禁止开放风向线、飞檐曲面、漂浮预测片、魔法阵、冰晶铠甲和成熟霸总式镜面礼装。
- 禁止把林野的窄矩形、平行切面或同心层复制给小铭与姜禾。
