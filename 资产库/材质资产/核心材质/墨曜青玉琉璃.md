# 墨曜青玉琉璃

> 小铭v04专属琉璃材质。只定义可测量的材质表现与安装范围，不覆盖小铭身份、礼装正史或道具功能。

## 物理属性

- 采用2.5—6毫米厚度，边缘倒角0.8—1.5毫米；任何视角都必须看见实体厚度。
- 表面粗糙度0.18—0.30，金属度0，基础透射20%—34%，折射率1.50—1.55。
- 内部为烟黑内部，保留低密度墨烟纵深与细小划痕；边缘呈青玉边缘，不是高饱和自发光描边。
- 折射必须随厚度、倒角、入射角和环境光变化，并在服装或地面上产生方向一致、边缘柔化的真实折射与投影。
- 允许少量指纹、细划痕和使用磨损；禁止无瑕亚克力板、液态糖浆或镜面铬。

## 固定结构与几何

- 由旧金／黑钛框架完整包边、夹持或铆接；旧金负责礼序节点，黑钛负责受力，不得无支撑悬浮。
- 几何服从小铭的门纹、山水等高线、流线与纵向层叠；转折处保留可制造倒角和真实连接缝。
- 榫卯磁吸只用于可见闭合逻辑，不得变成无接触漂浮机关。

## 使用范围

- 只用于斜襟、肩缘、前臂窗、腰封嵌片和鞋跟窗。
- 上述区域外不得扩散到脸、头发、整条四肢、胸腹主体、手掌、关节内侧、主织物或核心道具主体。
- 同一人物卡的正、侧、背视图必须保持位置、面积、厚度、框架和纹理连续。

## 光线响应

| 条件 | 固定响应 |
| --- | --- |
| 2800—3200K暖光 | 烟黑内部略显暖深灰，旧金节点可见但不橙化；青玉边缘只保留低亮冷色 |
| 4100—4300K身份光 | 烟黑层次、真实厚度、青玉边缘与旧金／黑钛框架同时可辨，不把肤色反射成青色 |
| 6800—7600K冷轮廓光 | 青玉边缘和内部折射增强，品红仅允许出现在极窄反射交界，不覆盖大面 |
| 干态 | 粗糙度与细划痕可见，投影清楚，不能像抛光塑料 |
| 湿态 | 高光增强约12%，基础透射降低约6%，水珠沿框架聚集且不抹去实体厚度 |
| 近景 | 显示倒角、烟黑内部、细划痕、框架夹持与连接缝 |
| 远景 | 只保留烟黑体块、青玉窄边和旧金／黑钛受力轮廓，不放大数据辉光 |

## 禁止项

- 禁止全身玻璃、果冻塑料和悬浮碎片。
- 禁止透明玻璃人、糖果水晶、彩虹油膜、无厚度贴片、无来源发光和重装机甲化。
- 禁止把小铭的门纹和青玉边缘复制给林野或姜禾。
