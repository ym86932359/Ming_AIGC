## 单一正式来源

1. 每个资产只有一个正式Markdown；其他文件只用稳定ID和双链调用。
2. 正式状态统一为`草案→待审→锁定→废弃`。
3. 锁定资产修改时递增版本，旧版保留历史记录，不覆盖图片。
4. 分镜不得引用“文件夹里看起来相似”的图片，必须从关联包或资产卡读取当前有效版本。
