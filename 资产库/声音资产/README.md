## 文件与分轨规则

| stem_id | 分轨 | 正式格式 | 内容 |
| --- | --- | --- | --- |
| `DX` | 对白 | 48kHz／24bit WAV，单声道源 | 日常、疯癫、蔑视及客人对白 |
| `FX` | 拟音 | 48kHz／24bit WAV | 门铃、杯、匙、布、脚步、灯闪和设备声 |
| `BG` | 环境 | 48kHz／24bit WAV，立体声或多声道 | 雨巷、酒馆房间、高架、旧车站和城市底噪 |
| `MX` | 音乐 | 48kHz／24bit WAV，立体声 | 原创配乐及授权音乐 |
| `FULL` | 全混 | 48kHz／24bit WAV | 对白、拟音、环境和音乐的审核母带 |

命名：`项目-集数-分轨-镜头或台词编号-状态-vNN.wav`。例如`ZERO-ECHO-PV01-DX-L06-FERAL-v02.wav`。
