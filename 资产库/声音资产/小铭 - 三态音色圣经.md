# 小铭 - 三态音色圣经

三种声音必须来自同一个本人声源。小丑状态是表演、剪辑与轻度声音处理形成的第二人格感，不得换成另一名配音者或怪兽变声。

## 三态参数

| voice_state_id | 日常酒保 | 疯癫小丑 | 蔑视小丑 |
| --- | --- | --- | --- |
| 编号 | `XM-VOICE-DAILY` | `XM-VOICE-FERAL` | `XM-VOICE-CONTEMPT` |
| 音高 | 本人自然中低区 | 同源低移0.5—1.5半音；保留共振 | 同源低移1—2半音；禁止厚重怪兽化 |
| 共振 | 胸腔与口腔平衡，近讲 | 前置共振略增，齿音受控 | 胸腔略增、声场收窄，口腔干净 |
| 语速 | 3.5—4字／秒 | 4—4.5字／秒，快—停—快 | 3—3.5字／秒，关键词前留白 |
| 气息 | 低、稳、少 | 短笑带气；耳语层8%—12% | 几乎无耳语，句尾收干 |
| 动态 | 峰均差自然，情绪不抢听众 | 峰值可高1—2dB，但笑声不过对白 | 动态收窄，压迫来自静止与靠近 |
| 空间 | 木质早反射，短尾 | 25—35ms轻双轨＋极短预回声 | 近乎干声，中央定位 |
| 禁止 | 广告腔、心理导师腔 | 连续尖笑、重失真、疯吼 | 咆哮、拖腔、低频轰鸣遮字 |

## 切换设计

```text
日常声尾
→ 环境抽空2—4帧
→ 80—120Hz低频脉冲一次
→ 2帧灯闪／画面硬切
→ 小丑辅声提前10—30ms进入
→ 主声在新妆造首帧清楚落字
```

声音变换不得早于画面超过2帧，也不得延后超过3帧；否则观众会把妆造切换理解为配音错位。回到日常状态时取消耳语与预回声，但保留同一房间早反射。

## 演员表演提示

- 日常：先听完再说；句首不要抢气，句尾给对方留空间。
- 疯癫：不是大声，而是更快、更近、更享受揭穿的瞬间；每段最多一次短笑。
- 蔑视：不笑、不歪头、不抬高音量；像答案显而易见，厌倦的是自欺而不是受伤的人。

## 审核句

| 状态 | 审核句 | 验证目标 |
| --- | --- | --- |
| 日常 | “这里不卖遗忘。” | 温和但不安慰式虚假保证 |
| 疯癫 | “酒哪有秘密好喝？” | 短笑、快—停—快、中文辅音清楚 |
| 蔑视 | “他陪你看完，我负责拆穿你。” | 静止、干、慢、同一身份可辨 |

## 技术母带

- 收音：48kHz／24bit WAV；峰值建议不高于-6dBFS，保留后期空间。
- 同一麦克风、距离、房间和前级完成三态录音；需要变化时由表演优先、处理其次。
- 每句保留三遍：技术干净版、表演版、口型清晰版；另录30秒房间底噪。
- 正式处理链必须保存可撤销参数，不能只交付压扁后的全混文件。
