# 零点回声酒馆 - 高风险镜头分层交付规范

## 最小分层

| layer_id | 层 | 内容 | 必须提供 |
| --- | --- | --- | --- |
| `PLATE-CLEAN` | 干净底板 | 无人物、无临时道具、无VFX的完整空间 | 首帧、尾帧、原分辨率静帧或序列 |
| `FG-OCCLUSION` | 前景遮挡 | 门框、杯、雨线、蒸汽、经过物 | 带Alpha或独立遮罩 |
| `CHAR-A/B/C` | 人物 | 日常／疯癫／蔑视独立层 | 原始输出、人物遮罩、首尾帧 |
| `PROP` | 交互道具 | 杯、匙、相机、存储环和手部接触 | 道具遮罩、接触阴影 |
| `MIRROR-GLASS` | 镜面／玻璃 | 反射、折射、冷凝与镜像 | 无异常基准＋异常版 |
| `LIGHT` | 光效 | 实景灯、轮廓、杯内故障青、曝光脉冲 | 可关闭独立层，不烘焙肤色 |
| `ATMOS` | 大气 | 雨、蒸汽、记忆雾、灰尘 | 深度顺序、速度和衰减 |
| `UI-TEXT` | 字幕／UI | 中文、片名、预测字段、AIGC标识 | 可编辑文字＋无字画面 |
| `DEPTH-MOTION` | 技术参考 | 深度、运动、光流和遮挡关系 | 高风险镜头必须保存 |

## 高风险镜头判定

- 同一人物双形态切换、镜面延迟、透明道具、手指精细接触、发丝穿过透明层、雨水覆盖脸、中文口型或三层以上遮挡中的任一项。
- 高风险镜头不得只交付压平MP4；至少保留源图／源视频、提示词、参考图、任务编号、模型版本、首尾帧和合成工程。

## 审片顺序

```text
身份与骨相
→ 空间几何
→ 手与道具接触
→ 衣料／头发／配饰
→ 镜面／玻璃／雨雾
→ 灯光与色彩
→ 中文口型与声音
→ 字幕、UI和交付规格
```

上游错误不能用下游特效遮盖：脸漂移先回人物层，空间跳变先回底板，口型错位先回表演／声音层。

## 版本记录

每次返修记录：`shot_id`、输入版本、实际错误、修正范围、输出版本、SHA、审核人和结论。失败版移入历史目录，禁止覆写。
