## 调用结构

```text
人物连续性编号
→ 妆造状态
→ 表情编号＋中文情绪名
→ 动作编号
→ 口型编号序列
→ 强度等级
→ 进入状态／离镜状态
```

示例：

```text
XIAOMING-S1-ZERO-BARTENDER
日常赛博酒保／E06 共情心疼／XM-ACT-LISTEN-02
M00-M05-M03-M00／P1微表演
进入：左手托杯／离镜：视线回到林野
```

## 强度合同

| 等级 | 名称 | 面部位移 | 身体幅度 | 使用范围 |
| --- | --- | --- | --- | --- |
| `P1` | 微表演 | 眉眼、泪膜、鼻翼和嘴角轻微变化 | 呼吸、重心和视线为主 | 倾听、犹豫、内心松动 |
| `P2` | 正常表演 | 清楚可辨但不过度 | 一次明确手势或重心变化 | 对话、边界、解释和选择 |
| `P3` | 情绪峰值 | 允许泪水、颌肌和呼吸失衡 | 一次峰值动作后必须回落 | 受伤、失控前兆、妆造异常 |

同一镜头只能有一个主表演变化。`P3`不得连续跨越两个以上镜头，也不得用动漫颜艺替代真人肌肉逻辑。

## 锁定条件

- [ ] 三人完成5—10秒身份回归测试：人物总卡→表情→口型→动作→原身份。
- [ ] 小铭两种妆造说同一句台词时保持同骨相、同牙列和同天然瞳色。
- [ ] 手部与道具接触、衣料、头发和配饰在动作中无穿插。
- [ ] 每段锁定动作保存首帧、中间峰值帧、末帧和失败版本说明。
