# 姜禾纪实相机

## 物理与识别

- 紧凑无反机身`128×78×55mm`，镜头收纳长`45mm`、工作长`68mm`。
- 哑光黑钛复合外壳，右手握柄有细磨损；无品牌Logo。
- 深棕黑形状记忆织物肩带宽`24mm`，从姜禾左肩斜向右腰。
- 一枚低亮环境感应点位于镜头座左下，静息不可见。

## 交互与状态

| 状态 | 动作 | 连续性 |
| --- | --- | --- |
| `CAMERA-READY` | 右手扶机身、左手托镜头 | 观察生活与旧车站记忆 |
| `CAMERA-SHOOTING` | 从`CAMERA-READY`抬至眼下，右手按一次快门后回到准备态 | 本集只在旧车站早餐拍摄一次；快门、手指和取景方向固定 |
| `CAMERA-LOWERED` | 被求婚打动时双手离开，相机垂到胸前 | 肩带受力真实，不瞬移 |
| `CAMERA-GRIPPED` | 感到窒息时右手握紧肩带 | 手指压力与呼吸同步 |
| `CAMERA-RESTING` | 开放式同行中自然贴在右腰 | 不在吃面时悬浮胸前 |
| `CAMERA-TABLE-REST` | 吃面前由姜禾双手放在右侧桌面，镜头朝下、肩带仍连接左肩 | 只用于坐下进食；离席前必须回到`CAMERA-RESTING` |

## 禁区

- 禁止发光镜头、武器化相机、奢侈品牌标识、肩带方向跳变和镜头长度随机变化。
- 不用相机遮住姜禾的核心表情；近景中相机只能作为手部情绪道具。
