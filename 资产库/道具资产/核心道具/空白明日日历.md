# 空白明日日历

## 物理与识别

- `140×190mm`柔性暖象牙电子纸页，边缘嵌`0.4mm`低亮电子墨水织线。
- 只显示标题“明天”，没有时间格、任务、成功率和终点标记。
- 页面可在存储环删除后从投影中落到吧台，厚度和投影阴影必须可见。

## 状态

| 状态 | 表现 | 下一状态 |
| --- | --- | --- |
| `CALENDAR-HIDDEN` | 行为预演尚未开始，页面收在吧台内 | `CALENDAR-REVEALED` |
| `CALENDAR-REVEALED` | SH022升起，只显示“明天”，等待林野输入 | `CALENDAR-NOTIFICATION` |
| `CALENDAR-NOTIFICATION` | 显示旧式精确日程并承载左侧通知式预演 | `CALENDAR-TWO-PATHS` |
| `CALENDAR-TWO-PATHS` | 邀请式预演加入后，两条细线向不同方向延伸 | `CALENDAR-BLANK` |
| `CALENDAR-BLANK` | 语录后线条熄灭，只保留空白页面 | 终态 |

## 禁区

- 禁止自动写出姜禾的回复、婚礼日期、成功结局或鸡汤长文。
- 电子纸不发强光，不像悬浮HUD或平板电脑。
