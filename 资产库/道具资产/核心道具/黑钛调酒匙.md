# 黑钛调酒匙

## 物理与识别

- 总长`280mm`，匙面`31×22mm`，螺旋柄直径`3mm`，柄端固定`13mm`圆环。
- 拉丝黑钛，重心位于柄中后段；只有边缘出现窄高光。
- 小铭固定右手使用，拇指与食指控制旋转，中指稳定轴线。

## 状态与交互

| 状态 | 摆位/动作 | 连续性 |
| --- | --- | --- |
| `SPOON-RACKED` | 吧台内侧磁吸槽，匙面向左 | 开场与收尾固定位置 |
| `SPOON-MIXING` | 右手低速顺时针两圈半 | 杯不移动，不敲击杯壁 |
| `SPOON-PAUSED` | 林野说出关键矛盾时停止半秒 | 是小铭倾听标识，不挥舞 |
| `SPOON-RETURNED` | 擦干后回到磁吸槽 | 柄端圆环方向不变 |

## 禁区

- 禁止变成魔杖、武器、发光棒、左手随机使用或长度跳变。
- 声音只允许轻金属摩擦与一次杯沿触碰，不做夸张铃音。
