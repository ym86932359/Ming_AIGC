## 使用规则

1. 生成与分镜优先读取人物关联包、场景README和道具资产卡；总索引只负责找到图片和判断状态。
2. `历史禁用`、`候选/实验`、`工程中间`图片不得作为自动选图源。
3. 新图必须先按内容命名，再进入对应角色、场景、道具、镜头或工程目录；禁止使用`image1`、`frame01`、`final-final`等无语义名称。
4. 同一内容出现多份时不直接删除，通过SHA重复组说明各自职责；确认无依赖后才能另行清理。
5. 真人身份参考属于受限素材，只服务同脸一致性，不进入公开宣传资产包。
