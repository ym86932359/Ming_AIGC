## 编号规则

`accessory_id`使用`ZERO-ECHO-ACC-角色-名称-NN`，例如：

```text
ZERO-ECHO-ACC-XIAOMING-WAIST-BUCKLE-01
ZERO-ECHO-ACC-LINYE-RING-ATTACHMENT-01
ZERO-ECHO-ACC-JIANGHE-CAMERA-STRAP-01
```

## 调用规则

1. 每件配饰记录数量、左右位置、尺寸、材质、闭合方式和不可消失镜头。
2. 人物总卡负责整体轮廓；配饰卡负责安装、受力、摆动、碰撞和局部特写。
3. 同一物件同时具有“道具”和“配饰接口”时，道具卡保存物件正史，配饰卡只保存人体挂接关系，禁止复制状态机。
4. 妆造切换不得改变固定配饰数量和左右位置；允许变化必须写入状态表。
5. 图生视频前必须进行转身、抬手、坐姿、俯身和持物五项穿插测试。

## 锁定条件

- [ ] 正侧背安装位置和绑定区域一致。
- [ ] 闭合、受力、摆动和碰撞范围可以执行。
- [ ] 与服装、头发、手部和剧情道具不存在穿插冲突。
- [ ] 镜头可见性、取下状态和不可消失镜头已登记。
