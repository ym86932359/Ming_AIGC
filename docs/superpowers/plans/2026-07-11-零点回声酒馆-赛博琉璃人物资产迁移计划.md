# 零点回声酒馆赛博琉璃人物资产迁移 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将小铭、林野、姜禾的正式人物模型统一迁移到`ZERO-ECHO-VIS-CYBER-GLAZE-01`赛博琉璃风格，重制8张v03图片并同步全部Obsidian框架与调用入口。

**Architecture:** 项目风格规范是唯一美术来源，角色卡只保存身份与角色差异；人物关联包通过`visual_style_id`和`style_profile`引用风格规范。先重制总卡并审核身份，再以总卡为最高优先级生成表情卡，最后归档v02并原子切换当前入口。

**Tech Stack:** Obsidian Markdown、YAML Frontmatter、Wikilinks、GPT Image 2多参考生成、PNG、Ruby YAML验证、只读双链扫描。

## Global Constraints

- 正式风格编号固定为`ZERO-ECHO-VIS-CYBER-GLAZE-01`。
- 风格公式固定为真实身份55%＋动漫结构20%＋琉璃材质15%＋电影炫光10%。
- 当前范围固定为小铭、林野、姜禾三名已有正式关联包角色。
- v03图片先为`待审`；用户确认前不得标记`锁定`。
- 小铭三张本人照片高于所有生成图；小铭日常与小丑必须同脸。
- 四张表情卡必须各含01—24编号与准确中文情绪名。
- v03通过技术审核前，不移动v02；切换后v02只进入`历史版本/`，不删除。
- 项目不是Git仓库，不执行提交、分支、合并或PR。
