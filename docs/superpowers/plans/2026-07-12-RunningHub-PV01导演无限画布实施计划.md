# RunningHub PV01导演无限画布实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把RunningHub画布`2075968891168833538`建设为《零点回声酒馆》PV01可直接进入动态镜头生产的导演作战墙。

**Architecture:** 采用横向导演流水线和纵向资产分层。正史资产只上传一次，14镜时间轴负责阅读，16个Seedance任务负责执行，POST-E01负责片尾艺术字，候选Take、接受Take与废弃素材分区管理。

**Tech Stack:** RunningHub无限画布、浏览器文件上传、Obsidian正史资产、Seedance 2.0序列合同、PNG/WAV/MP4媒体。

**执行记录（2026-07-12）：** 用户启用Chrome文件网址访问后恢复执行。53项源媒体、10张导演信息板和1个导演总控文本节点已进入目标画布；网格吸附开启，空白节点清除，上传前后RH币余额均为`10.26`，未触发任何生成或付费工作流。

**画布适配说明：** RunningHub会按上传批次自动排布大量媒体。当前外部画布仍保留已验收的v01信息板；本地v02把14镜、16个Seedance任务与POST-E01后期任务重新排成10张1920×1080导演信息板。依赖线使用卡内`参考职责／不得继承／结束状态`合同，避免跨区连线穿过人物与分镜画面。本地v02尚未替换外部画布，不得误记为已上传。

## Global Constraints

- 使用目标画布：`https://rhtv.runninghub.cn/projects/canvas/2075968891168833538`。
- 人物只使用根目录正式v04和已锁定A/B/C板，不上传候选版、历史版或三张真人原始照片。
- PV01当前可调用世界／场景／道具为WK01—WK05、KF01—KF04、KF06、KF08、PROP01—PROP07。
- KF05、KF07只登记禁用说明，不上传为生成参考。
- 共上传53项媒体：29张正史／表演图、4张分镜板、18条音频、1条导演预演、1张片尾海报。
- 建立14张镜头卡、16张Seedance任务卡、1张POST-E01后期卡、C01—C08序列地图与审片状态机。
- 不启动AI生成、不运行付费工作流、不消耗RH币。
- 导演预演必须标为`ANIMATIC ONLY`，不得标成最终动态成片。
- 任何上传失败都先记录失败文件与原因，再单独重试；不得重复上传已经成功的同名资产。
