# PV01电影级锁片与片尾艺术字 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把《零点回声酒馆》PV01从60秒导演分镜预演升级为64秒电影级锁片预演，修复全部声画转场、人格切换、对白时序与片尾识别问题，并同步Obsidian生产资料。

**Architecture:** 60秒故事主体保持十四个叙事段落，但用一份13组镜间转场合同统一摄影、动作、灯光、声音与连续性；S10→S11承担唯一完整人格夺权特效，S13→S14改为同一连续镜头的内部表演节拍。60秒硬切黑场后新增4秒独立片尾艺术字子合成，使用既有烟木黑、旧黄铜、暖象牙和左上琥珀光，不改项目视觉正史。

**Tech Stack:** Obsidian Markdown/YAML/Wikilinks、Seedance 2.0序列与I2V/FLF2V提示词、HyperFrames HTML子合成、GSAP 3.14、FFmpeg/FFprobe、RunningHub SVG/PNG信息板。

## Global Constraints

- 60秒故事内容与13句对白不改写；正式预演总长扩展为64秒。
- 日常、小丑疯癫、小丑蔑视始终是同一小铭身份，不得融脸、半脸拼接或随机换脸。
- S10→S11使用“暖亮→暗→暖亮→暗→酒红显影→昏暗稳定”的唯一完整人格夺权切换。
- 酒红帧只短暂显出精准重合的小丑面孔；稳定后回到低照度琥珀主光与冷青轮廓，不维持满屏红光。
- 狰狞低笑为独立后景声层；对白保持中置，低笑不得遮蔽台词或造成单声道相位抵消。
- S12→S13继续采用无闪硬切；S13→S14不发生可见剪切。
- 60秒整硬切纯黑；片尾艺术字依次显示“零号回声酒馆”“敬请期待”，由左上方琥珀光擦亮。
- 当前保留历史Take01与Take02：Take01为技术废弃，Take02技术通过但改编为S02入口源片；二者都不得误标为S01正史成片。S01正式世界空镜与其余动态镜头仍待生成。
- 不覆盖v01文件；新输出统一递增为v02。
