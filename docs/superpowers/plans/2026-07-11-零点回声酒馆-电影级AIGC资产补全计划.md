# 零点回声酒馆电影级AIGC资产补全 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将《零点回声酒馆》从已完成的视觉开发和60秒动态分镜预演，推进到人物、场景、道具、配饰、声音和合成均可追溯的电影级AIGC生产状态，并用20—30秒PV01金样片验证整条生产链。

**Architecture:** [[个人创作/ai铭仔/资产库/零点回声酒馆 - 电影级AIGC资产补全总清单|补全总清单]]是状态唯一来源；`资产库`保存跨镜正式资产，人物关联包保存当前人物版本，PV01和EP01只保存逐镜调用与制作输出。任何候选资产先在候选/历史区域完成审核，再通过关联包原子切换，禁止在分镜中临时指定另一版本。

**Tech Stack:** Obsidian Markdown、YAML Frontmatter、Wikilinks、PNG/JPEG、WAV 48kHz/24bit、H.264/H.265、Seedance 2.0、HyperFrames、FFmpeg/FFprobe、`rg`、`shasum`、`sips`。

## Global Constraints

- 当前人物生产源已一次性切换为`ZERO-ECHO-VIS-NEO-CHINESE-CYBER-GLAZE-02`与v04；v03与`v04-cg02`只作历史追溯。
- 正式人物资产卡目标为3:4；平台实际返回2:3时记录真实尺寸，不拉伸、不伪报3:4。
- 剧情和宣传片母版固定16:9、24fps；金样片优先3840×2160，最低1920×1080。
- 小铭必须使用本人身份和经授权声音，不得使用随机脸或未经授权音色进入发布版。
- 当前60秒MP4是动态分镜预演，不标记为Seedance最终成片。
- 所有资产状态只使用`草案→待审→锁定→废弃`；设计状态和视觉状态分开记录。
- 历史、失败和候选文件不删除、不覆盖；正式入口只指向一个当前版本。
- 图片和Markdown修改使用`apply_patch`或相应图像生成/编辑流程，不使用破坏性覆盖。
- 不同步个人网站、不部署阿里云、不修改《月陨》等其他项目。
- 当前目录不是Git仓库；用SHA、验证日志和清单勾选代替提交记录。
