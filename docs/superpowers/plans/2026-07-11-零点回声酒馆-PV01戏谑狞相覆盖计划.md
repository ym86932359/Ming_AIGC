> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将已确认的“疯癫毒舌对话态＋沉稳蔑视独白态＋宣传片无镜直接切换”覆盖到PV01正式制作包，并建立人物表演与正片镜中母题接口。

**Architecture:** 以已审核设计稿为唯一改写源，先原子更新导演方案、14镜执行表和8段序列，再同步前置总包与视觉资产调用状态，最后新增人物表演接口并做全库一致性校验。现有20张视觉母资产不删除；KF05、KF07退出PV01当前调用并转为正片镜中母题历史参考。

**Tech Stack:** Markdown、YAML Frontmatter、Obsidian Wikilinks、Ruby 2.6兼容校验脚本、macOS文件系统。

## Global Constraints

- 宣传片固定60秒、16:9、3840×2160、24fps。
- 宣传片中不得使用镜中小丑、镜像对话或镜子完成变身。
- 日常与小丑使用同一连续性编号 `XIAOMING-S1-ZERO-BARTENDER`。
- 小丑必须区分 `PV01-XM-CLOWN-FERAL` 与 `PV01-XM-CLOWN-CONTEMPT`。
- 日常/小丑只用同机位A/B板硬切，禁止连续融脸。
- 现有图片不得删除、覆盖或移动；只调整调用状态。
- EP01现有240秒时码不得修改。
- 当前目录不是Git仓库，不执行提交、分支或工作树操作。
