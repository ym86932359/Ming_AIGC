# 小铭赛博琉璃画风实验 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 生成一张保持小铭真人身份的3:4“赛博琉璃”人物卡画风预览，并将提示词和实验状态写入Obsidian。

**Architecture:** 三张本人照片负责身份锚定，当时的v02人物总卡负责发型、体态和酒保轮廓，新提示词只改变表面画风、琉璃材质和灯光。实验资产放入独立目录，不修改当时的正式人物模型关联包版本。

**Tech Stack:** Obsidian Markdown、Wikilinks、GPT Image 2多参考图像编辑、PNG。

## Global Constraints

- 只生成小铭日常赛博酒保形态，不加入小丑妆。
- 保留本人脸型、五官比例、发际线、天然瞳色与成年身份。
- 画风采用写实基底、动漫锐线、虹彩琉璃材质与青紫电影炫光。
- 面部保持中性柔光，科技炫光不覆盖五官。
- 结果状态为`实验待审`，不得替换正式v02人物资产。
