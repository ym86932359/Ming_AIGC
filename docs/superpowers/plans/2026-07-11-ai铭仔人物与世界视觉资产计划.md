# ai铭仔人物与世界视觉资产 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. This is a tightly coupled Obsidian creative-asset task and is executed inline.

**Goal:** 为《零点回声酒馆》建立可复用的人物模型资料、世界观环境视觉圣经和16:9酒馆环境基准。

**Architecture:** 通用人物模板继续以 `个人创作/ai漫剧_提示词资料库` 为唯一来源；`ai铭仔` 只保存项目实例。人物身份、环境视觉和生成提示词分开维护，通过项目README和Obsidian双链汇合。

**Tech Stack:** Obsidian Markdown、YAML Frontmatter、Wikilinks、GPT Image 2提示词、16:9电影概念图。

## Global Constraints

- 人物总资产卡采用3:4；环境概念图和剧情分镜采用16:9。
- 小铭不得在没有用户本人身份参考图时生成随机正脸。
- 人物总卡不包含导演、镜头、景别或场景拆解。
- 科幻表现克制，只通过记忆酒液、镜面延迟、青蓝电路和城市基础设施显露。
- 正式图片均先为“待审”，人工确认后才能“锁定”。
- 所有方括号模板变量必须在项目实例中消除。
