# 《未写完的未来》单集剧本实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. This task is executed inline because the deliverable is one tightly coupled creative script.

**Goal:** 完成一集约 3 分钟、可直接进入分镜和 AIGC 制作的《零点回声酒馆》单集剧本，以故事表达“不要替未知的未来许下虚假保证，未知本身值得期待”。

**Architecture:** 采用栏目固定结构：灵魂拷问、客人入席、第一次主观记忆、第二次看见、现实小行动、酒馆暗线、妆后小铭语录。情感冲突由“以确定性控制未来”和“在变化中持续选择”构成，科幻视觉由未来模拟装置承担。

**Production Stack:** 16:9电影级横屏母版、真人小铭表演、AI赛博转化、AI客人及记忆世界、真实鸡尾酒、约180–200秒成片；9:16仅作宣发切片。

## Global Constraints

- 客人必须称呼酒保为“小铭”。
- 开头只提出灵魂拷问，不提前公布完整语录。
- 小铭不诊断、不保证治愈，只通过提问、复述与小行动推动成长。
- 小铭在尾声才换上完整赛博小丑妆并说出唯一核心语录。
- 真实鸡尾酒一客一杯、不续杯，不能被描绘成解决问题的方法。
- AIGC 奇观只服务情绪转折，现实沟通保留真人停顿和微表演。
- 结尾语录后保留静默，不口播点赞关注。
