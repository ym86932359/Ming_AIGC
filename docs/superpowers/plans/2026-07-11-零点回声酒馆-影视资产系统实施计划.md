# 零点回声酒馆影视资产系统 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking. This Obsidian vault is not a Git repository, so validation records replace commit steps.

**Goal:** 建立可供长篇AIGC连续剧复用的色卡、场景、道具、材质和人物接口资产库，并把EP01升级为230—240秒的完整制作包、逐镜头导演分镜及表演连续性资料。

**Architecture:** 所有跨集资产只在`资产库`或`人物资产`保存一次，分集制作包通过稳定编号和Obsidian双链调用。完整剧本负责故事正史，导演分镜负责每镜头画面与技术参数，表演连续性表负责微表情和肢体状态，场景色卡与道具表负责跨镜头状态。

**Tech Stack:** Obsidian Markdown、YAML Frontmatter、Wikilinks、Mermaid、GPT Image提示词规范、16:9电影级AIGC生产字段。

## Global Constraints

- 剧情母版固定为3840×2160或等比例16:9；人物总资产卡固定为3:4。
- 统一画风为“年轻化近未来仿真人情感现实主义”，约70%现实可信度、30%近未来识别度。
- 林野正式年龄26岁，视觉年龄25—27岁；姜禾正式年龄24岁，视觉年龄23—25岁。
- 小铭必须服从本人照片并只做约5岁克制年轻化，禁止随机换脸和幼态化。
- 全局色卡、场景、道具和材质均采用“草案→待审→锁定→废弃”。
- 资产编号一经进入正式分镜不得复用；历史图片不覆盖、不删除。
- 每个镜头必须包含表演、场景、色光、道具、声音、特效、提示词和连续性字段。
- 右侧开放式约会画面只标记为“一种可能”，不得保证姜禾一定回来。
- 不同步网站、不部署服务、不修改`月陨`等其他项目。
