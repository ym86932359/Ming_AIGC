# PV01石砖片尾分镜与GPT Image视觉板 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为PV01的60—64秒石砖场景化片尾建立可拍、可生成、可进入HyperFrames制作的导演分镜剧本，并用GPT Image 2生成一张同墙体连续的四格电影视觉板。

**Architecture:** 先在Obsidian建立片尾资产目录、分镜剧本和GPT Image精确提示词，再把四个时码节拍压缩进一张2×2视觉板。四格共享同一面烟黑玄武岩承重墙、同一修葺字砖、同一2800K左上掠射光；视觉板只验证空间、砖字、光位和运镜端点，不替代最终64秒动态成片。

**Tech Stack:** Obsidian Markdown/YAML/Wikilinks、GPT Image 2平台托管生成、16:9电影分镜、精确简体中文、HyperFrames后续合成。

## Global Constraints

- 保留60秒正片与13句对白，不修改S01—S14内容。
- 片尾文字固定为“零号回声酒馆”与“敬请期待”，不得改字、少字、增字、使用英文或拼音。
- 两层文字都必须由实体砖、砂浆、深度和阴影构成，不使用悬浮字幕、金属艺术字、霓虹或投影。
- 主标题终帧占画面宽度82%—88%；“敬请期待”使用较小凹陷砖石镶嵌。
- 四格必须是同一面墙、同一砖纹、同一字形、同一光向，只有镜头距离和揭示范围变化。
- 输出采用16:9宽银幕、高质量、不含人物、品牌Logo、水印、二维码或额外文字。
- 平台托管图像生成完成后，只有取得平台返回的原始PNG路径并验证哈希一致，才允许复制进Obsidian；不得用会话截图冒充原图。

- [x] **Step 3: 锁定电影摄影与世界观**
