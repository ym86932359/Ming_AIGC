# PV01记忆回声渗光v06生成 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在不改变v05文字、排版、墙体和主光的前提下，增加世界观一致的墙后“记忆回声渗光”并生成v06审片样张。

**Architecture:** 以v05为唯一参考图，通过GPT Image精确编辑只增加6—8处砂浆缝透光、最多两束体积微光和对应浮尘。v06必须重新通过六字、无标点、副标题、字距、墙体连续性和渗光层级审核；任何文字漂移都废弃v06并回到v05重新生成。

**Tech Stack:** Obsidian Markdown、GPT Image 2平台参考图编辑、macOS Vision文字识别、`sips`、SHA-256。

## Global Constraints

- 唯一参考图为`PV01-POST-E01-石砖终帧母图-v05.png`。
- 只增加墙后渗光、最多两束微弱体积光和少量浮尘。
- 主标题仍为六字`零号回声酒馆`，无标点、无重复“声”。
- 副标题仍为`敬请期待`。
- 六字字形、字距、基线、厚度、位置与石材纹理不变。
- 墙体、裂缝、砂浆、锚钉、梁柱、地面、构图和曝光不变。
- 左上3000K主光、右下投影和原文字背光不变。
- 只允许6—8处短缝发光，暖象牙／琥珀85%、故障青15%、总面积不超过2%。
- 体积微光不超过两束，静态亮度代表4.8秒呼吸周期的55%状态。
- v06失败时不以失败图作为下一版参考。
- 多角度资产继续阻断。
- 当前目录不是Git仓库，不执行commit、merge或PR。
