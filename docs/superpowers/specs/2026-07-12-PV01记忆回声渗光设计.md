# PV01｜记忆回声渗光设计

## 一、叙事定义

“记忆回声渗光”来自石砖承重墙背后的零号酒馆记忆储存层。客人的情绪与记忆被酒液唤醒后，不会完全消失，而会以低强度光信号进入建筑结构，在墙后缓慢流动并从砂浆缝、修补缝和字后空腔中透出。

这层光不是霓虹装饰、魔法符文或故障特效，而是“这面墙后面仍有记忆在呼吸”的物理证据。它服务世界观暗线：酒馆收纳过的情绪正在悄悄改变建筑，也预示小铭体内出现无法清除的记忆残留。

## 二、v06唯一改动

以已通过导演技术审片的v05单一终帧为唯一图像参考，v06只增加墙后渗光，不重建设计。

### 必须保持不变

- 主标题严格为六字“零号回声酒馆”，无逗号、无标点、无重复“声”。
- 副标题严格为“敬请期待”。
- 六字字形、字距、基线、厚度、石材纹理和位置不变。
- 墙体砖阵、裂缝、砂浆、盐渍、锚钉、梁柱、地面和构图不变。
- 左上3000K琥珀主光、右下实体投影和现有文字背光不变。
- 画幅、镜头、曝光、颗粒和世界观色卡不变。

### 只允许增加

- 既有墙缝中的低强度透光。
- 两束以内、近乎不可见的体积微光。
- 与微光相匹配的少量浮尘和近邻石面反射。

## 三、渗光位置

全墙只允许6—8处短墙缝亮起，不形成连续光网。

| 区域 | 数量 | 作用 |
| --- | ---: | --- |
| “回—声”字后与两字之间的既有砂浆缝 | 3—4处 | 作为“回声”概念的主要叙事光源 |
| 主标题右下方、靠近“酒—馆”下缘 | 1—2处 | 与文字右下投影发生视觉呼应 |
| 副标题上方或两侧既有墙缝 | 1—2处 | 把主标题与“敬请期待”连接为同一空间能量 |

- 不新增裂缝，不改变砖块位置。
- 不让光线穿过汉字笔画内部，不影响六字可读性。
- 不在墙体四周平均撒光，避免变成装饰灯带。

## 四、颜色与亮度

| 光层 | 色彩 | 占比 | 亮度边界 |
| --- | --- | ---: | --- |
| 记忆暖芯 | 暖象牙至低饱和琥珀 | 85% | 不超过主标题最亮石面的8% |
| AI信号边缘 | 故障青 | 15% | 只在暖芯边缘出现，不形成独立青线 |

- 暖芯建议范围：`#D5CDC1`至`#B9793A`的低亮混合。
- 故障青沿用`#59AAB4`，饱和度和亮度均低于副标题青边。
- 渗光总面积不超过画面的2%。
- 墙体暗部保持烟木黑和雨夜蓝灰，不能被渗光整体抬亮。

## 五、光学表现

### 砂浆缝透光

- 光从墙后穿过砂浆孔隙，缝中心较亮、边缘快速衰减。
- 单条亮缝长度不超过相邻砖块宽度的1.5倍，宽度保持细窄。
- 光只照亮近邻1—3厘米石面，保留石材毛孔和砂浆颗粒。

### 体积微光

- 最多两束，分别从“回—声”附近与副标题上方的深缝斜向透出。
- 光束长度短、边缘柔和、透明度极低，只在浮尘通过时可见。
- 不出现舞台追光、放射光芒、镜头光晕或雾化光柱。

### 与现有光影关系

- 左上琥珀主光继续决定石材明暗和右下实体投影。
- 字背光继续只服务汉字轮廓。
- 记忆渗光属于墙后次级光源，必须被墙体遮挡并沿缝出现；不能改变主标题的主光方向。

## 六、动画连续性

静态v06母图显示渗光处于约55%的呼吸亮度，既能被看见又不处于峰值。

| 参数 | 锁定值 |
| --- | --- |
| 呼吸周期 | 4.8秒 |
| 亮度范围 | 静息45%—峰值65% |
| 波形 | 缓慢正弦式起伏，无突然跳变 |
| 色相变化 | 暖芯不变；故障青只在峰值前后增加约5% |
| 副标题触发 | “敬请期待”显现时，墙后渗光整体上升约8%，0.6秒后回落 |
| 闪烁 | 禁止高频闪动、酒红闪动和随机局部熄灭 |

未来A01—A04角度图必须继承同一组6—8处渗光位置。相机角度改变时，亮缝透视、近邻反射和光束遮挡可以变化，但墙缝坐标、颜色比例和呼吸相位不能随机重设。

## 七、生成策略

### v06生成

- 模式：参考图精确编辑。
- 唯一参考：v05母图审片样张。
- 只改变墙缝透光、两束以内体积微光和对应浮尘。
- 不允许重新生成文字、墙体或构图。
- 输出递增为v06，不覆盖v05。

### 失败处理

出现以下任一情况，v06直接废弃，不继续局部修字：

- 六字发生错字、增字、少字、标点或字距漂移。
- “声”字变形或出现第二个“声”。
- 副标题改变。
- 渗光变成霓虹灯带、魔法符文或满墙光网。
- 墙体、锚钉、梁柱、地面或主光方向漂移。

失败时重新以v05为参考，只调整渗光提示词，不以失败v06作为下一版参考。

## 八、验收标准

- v05的文字、排版、构图、材质和光影连续性保持。
- 6—8处渗光集中在“回—声”、主标题右下和副标题附近。
- 暖象牙／琥珀约85%，故障青约15%，总面积不超过2%。
- 体积微光不超过两束，弱而可辨。
- 光源明确来自墙后，不像字体自身霓虹。
- 主标题仍是画面第一视觉层，副标题第二，渗光第三。
- 静态图可直接作为未来4.8秒呼吸动画的55%亮度参考帧。
