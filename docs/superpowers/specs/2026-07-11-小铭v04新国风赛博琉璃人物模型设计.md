# 小铭 v04 - 新国风赛博琉璃人物模型设计

## 一、目标与边界

以现有小铭本人身份、`XIAOMING-S1-ZERO-BARTENDER`连续性和零号酒保职业为不可变正史，设计一张“小铭 v04 新国风赛博琉璃人物资产母样”。本轮用于验证人物新画风，不立即替换当前v03人物总卡、妆造卡、双表情库或关联包。

已确认采用“新国风赛博琉璃”方向：

> 真实演员身份与人体结构为核心；新国风负责服装结构和文化秩序；动漫化3D CG负责轮廓、材质分层与电影视觉冲击；科技必须能解释酒保职业动作，不能把人物改造成仙侠、古装角色或重型机甲战士。

### 本轮包含

- 一张小铭日常零号酒保v04人物资产母样。
- 一份母样生成提示词和参考图职责表。
- 一份技术与人工审核记录。
- 母样在独立实验路径中的索引与版本状态。

### 本轮不包含

- 不生成v04赛博小丑妆造卡。
- 不生成v04日常或小丑24格表情卡。
- 不修改林野、姜禾或其他人物。
- 不切换人物关联包当前有效图片。
- 不移动、不覆盖、不删除现有v03资产。
- 不同步网站、不部署服务。

### 世界边界

- 城市生活、人的情感关系和酒馆服务仍然真实可识别；科技是日常工具，不是魔法。
- 国风来自斜襟、叠领、榫卯受力、漆器黑、青玉琉璃、旧金属、山水留白和传统建筑比例，不依赖龙凤贴图、仙侠法器或古装复制。
- 人物脸、眼、鼻、唇、皮肤、手和脚保持仿真人摄影质感。
- 动漫感只进入服装外轮廓、硬表面切面、层次阴影和动作剪影。
- 3D CG负责琉璃折射、织物纤维、金属磨损、道具结构和空间光影，不把皮肤渲染成塑料。
- 最终观感应是“真实演员进入高规格东方科幻电影”，而非动漫脸、真人皮或国风COS照。

## 四、人物模型细节

### 脸与五官

- 柔和偏长鹅蛋脸；额头平整开阔；颧骨不外扩；下颌从耳下逐渐内收；下巴小而圆。
- 深色自然毛流、低弧度平眉；自然杏仁眼，眼尾轻微上扬；真实眼裂、眼距和眼睑厚度。
- 天然深棕虹膜、黑色圆瞳、自然眼白、真实角膜高光与泪膜；无发光眼、无整眼灌色。
- 鼻梁纤细顺直，鼻尖小而圆润，鼻翼窄而自然；上唇唇峰清楚，下唇略饱满。
- 保留毛孔、绒毛、轻微油脂分区、唇纹和自然不对称；不磨成瓷娃娃皮肤。
- 视觉年龄只在本人参考基础上克制年轻约五岁；不放大眼睛、不缩短中庭、不改鼻翼、下颌或种族特征。

### 发型

- 深黑色中长层次发，近中分，额前两侧为自然帘式碎发。
- 耳前耳后层次清楚，颈侧发尾略向外翻；背面发量真实、发束闭合，不蓬成假发。
- 背面增加一枚隐藏式“墨玉发束芯”：拉丝黑钛外框、烟青玉内芯，只固定后颈发层，不改变发际线、长度或主轮廓。
- 头发保持真实发丝、粗细、飞发和微弱油脂光泽；动漫锐线不得把头发变成整块塑料片。

### 身体、手与脚

- 清瘦修长的成年青年体态，肩线自然、肌肉量克制；脊柱直而不军姿化。
- 倾听时上身轻微前倾，肩颈放松；不交叉双臂，不以手指居高临下地点人。
- 右手为调酒惯用手：拇指、食指与中指控制调酒匙旋转，无名指和小指稳定节奏。
- 左手托住记忆杯下半部，手掌、虎口、指节、甲床、指甲和玻璃接触压力真实。
- 双手均为五指，长度、关节方向和左右手正确；禁止粘连、增指、断指、橡胶手和无受力持物。
- 鞋楦服从真实脚掌、足弓、踝关节和脚趾空间；站姿有明确承重点，短靴弯折与鞋底压力符合真人步态。
- 资产卡显示短靴正面、外侧、背面、鞋底和鞋跟；不得用悬浮脚、芭蕾脚尖或错误内外翻代替足部连续性。

## 五、新国风零号酒保礼装

### 总轮廓

- 双层非对称斜襟构成第一识别线，外层从右肩下行至左腰，内层保持酒保动作所需的贴身与稳定。
- 半云肩式柔性护片形成肩部水平秩序，但只到肩峰与肩胛外缘，不扩大成戏服云肩或战甲。
- 窄袖与可调节腕口服务调酒动作；多层腰封固定道具；前短后长双开衩保证迈步、转身和弯腰。
- 下装为宽松但利落的黑色机能裤，配低跟智能防滑短靴；不使用长袍拖地、裙摆或盔甲下装。

### 结构与闭合

| 区域 | 新国风结构 | 物理闭合要求 |
| --- | --- | --- |
| 领口与斜襟 | 交领比例转译为非对称双层酒保领 | 隐藏磁吸扣＋榫卯式黑钛卡件，正侧背可追踪 |
| 半云肩 | 柔性形状记忆织物＋薄型墨曜琉璃 | 与肩线和肩胛活动方向一致，不悬浮 |
| 胸背 | 烟墨织物主体，山水等高线只在斜光下显现 | 后背中轴、肩片和侧缝闭合成立 |
| 前臂 | 窄幅黑钛骨架＋琉璃观察窗 | 腕关节弯曲时不穿模、不像发光手铐 |
| 腰封 | 多层织物、旧金圆扣和榫卯快拆结构 | 调酒匙、杯巾和红穗挂点固定 |
| 下摆 | 前短后长、后部双开衩 | 走动时分片合理，静止时恢复闭合 |
| 短靴 | 哑光皮革主体＋鞋跟微型状态窗 | 正侧背、鞋底、防滑纹和受力一致 |

### 图案与国风表达

- 只使用低对比回纹、电路化榫卯节点和抽象山水等高线；图案在正常光下接近不可见，在斜向轮廓光下才出现。
- 禁止直接使用龙、凤、麒麟、仙鹤、巨大祥云、古文字堆砌和随机假汉字。
- 国风识别来自比例、结构、留白和材质，不来自贴图数量。

## 六、材质与道具

### 材质系统

| 材质 | 使用区域 | 视觉参数 | 禁区 |
| --- | --- | --- | --- |
| 形状记忆织物 | 内衫、长衣、裤装、腰封 | 哑光、真实纤维、折痕可恢复 | 塑料胶衣、无重力飘带 |
| 拉丝黑钛 | 榫卯卡件、腕部骨架、道具结构 | 细拉丝、边缘磨损、低金属高光 | 镜面铬、厚重战甲 |
| 旧黄铜 | 领口、腰扣、滚边与固定点 | 暖旧金、细划痕、轻氧化 | 亮金网游装饰 |
| 墨曜青玉琉璃 | 斜襟、肩缘、前臂观察窗、鞋跟状态窗 | 2.5—6毫米厚度、烟黑内部、青玉边缘折射、真实投影 | 悬浮碎片、全身玻璃、果冻塑料 |
| 电子墨水织线 | 暗纹与状态提示 | 静息低亮、斜光可见 | 常亮HUD、满身霓虹 |

### 核心道具升级

- **黑钛调酒匙**：维持约28厘米长柄与右手调用；增加榫卯式握柄和“玉衡配重珠”，使旋转重心可见。不得变成权杖、剑或光棒。
- **未定式记忆杯**：保持透明厚壁低杯；内壁形成低亮山水雾纹，青蓝记忆雾不超过杯体三分之一。不得改成茶盏、酒樽、法器或漂浮全息球。
- **酒红擦杯巾**：深酒红形状记忆织物，折叠方式固定；用于开场与送别动作，不扩大成披风。

### 灯光合同

- 主光：左前约35°、俯角20°、4300K大型柔光，保护脸型、五官、毛孔和自然肤色。
- 辅光：右前低强度5200K柔光，主辅比约4:1，确保黑衣、双手与道具暗部可读。
- 轮廓光：左后7200K青玉窄光，只描发丝、琉璃厚度和外轮廓。
- 品红折射：只在琉璃斜角反射中出现，画面占比低于2%，不得污染皮肤、眼白、天然虹膜和唇色。
- 眼神光：正前略高的小型柔光点；天然深棕虹膜、黑瞳、眼白与泪膜清楚。
- 琉璃高光上限必须保留内部深度和固定框架，不剪切成白片或霓虹边。

## 九、生成合同

### 生成模式

- 请求类型：多参考图人物资产卡生成／参考编辑。
- 质量：高质量；中文与多面板细节优先。
- 目标画布：3:4竖版；输出实际比例与像素必须记录，不允许为满足比例强行拉伸人物。
- 产物数量：首轮只生成一张母样；不同时生成多个发散版本。

### 提示词顺序

1. 先声明3:4竖版画布与四个固定区域。
2. 再逐张声明参考图职责和不可继承内容。
3. 锁定同一成年小铭及五项身份锚点。
4. 描述身体、手脚、服装、道具与物理受力。
5. 分开描述材质、色卡和灯光。
6. 列出准确中文短标签。
7. 最后使用短而有针对性的禁止项保护身份与结构。

### 核心不变量

- 同一脸、同一成年视觉年龄、同一天然虹膜、同一发际线和同一深黑中长发。
- 主形象与三视图保持同一肩腰比例、服装闭合、配饰数量、鞋履和道具尺寸。
- 右手调酒匙、左手记忆杯；不得左右互换。
- 新国风只改变结构和材质表达，不改人物职业、身份或核心道具类别。

### 重点禁止

- 动漫大眼、二维脸、网红尖下巴、欧美化、幼态化、塑料磨皮和随机换脸。
- 仙侠、道士、汉服COS、马戏小丑、戏服、重装机甲、全机械四肢和战士化。
- 龙凤贴图、巨型祥云、随机假汉字、品牌Logo、密集HUD和满屏霓虹。
- 错误手指、无受力持物、悬浮脚、裁头、裁手、裁脚和三视图服装跳变。
- 琉璃塑料化、悬浮碎片、全身透明、青紫染脸、发光眼和道具光剑化。
