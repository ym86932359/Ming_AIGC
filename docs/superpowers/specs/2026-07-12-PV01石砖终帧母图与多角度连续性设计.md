# PV01｜石砖终帧母图与多角度连续性设计

## 一、设计结论

废止“在同一张四格板中重复生成完整中文”的路线。新流程先从空白构图生成一张唯一的16:9终帧母图，待文字、排版、材质与光影全部通过后，再只以该母图作为视觉身份源派生不同角度和近景。历史v01—v04不得作为新图参考，避免重复字、错误字位、异常空槽和旧墙几何继续传递。

终帧只保留两层准确文案：

```text
零号回声酒馆
敬请期待
```

- 主标题严格为六个简体中文大字，无逗号、句号或其他标点。
- 副标题严格为四字“敬请期待”。
- 不出现英文、拼音、Logo、水印、二维码或额外可读文字。

## 二、单一终帧母图

### 输出规格

| 项目 | 锁定值 |
| --- | --- |
| 画幅 | 16:9横屏 |
| 母版目标 | 3840×2160 PNG |
| 构图 | 单一完整终帧，不做四格、不重复标题 |
| 镜头 | 32mm建筑宽景，机位约1.45米，轻微低机位仰角2°—3° |
| 场景 | 零号酒馆与旧城高架共用的地下承重结构层 |
| 生成模式 | 从零生成，不附带v01—v04参考图 |

### 墙体结构

- 主墙为烟黑玄武岩承重墙，雨夜蓝灰暗部、暖灰矿物砂浆、潮湿盐渍、少量氧化旧黄铜锚钉。
- 墙体中央为完整文字安装区域，周围保留足够负空间，不让梁柱、锚钉或裂缝穿过汉字主笔画。
- 砖块尺度、裂缝位置、盐渍流向、锚钉坐标、地面反射和顶部梁结构在母图通过后全部锁定，作为多角度连续性锚点。

## 三、文字与排版合同

### 主标题

| 字位 | 字符 | 字形要求 |
| --- | --- | --- |
| 1 | 零 | 雨字头与下部结构清楚，不得像“翠”“霜”或数字 |
| 2 | 号 | 上部“口”和下部结构明确，不得像“考” |
| 3 | 回 | 外框、内口完整，不能被砖缝切断 |
| 4 | 声 | 标准简体“声”，上部“士”和下部结构清楚；不得重复、不得像“笋”“青” |
| 5 | 酒 | 三点水与“酉”左右关系明确，不得与“声”粘连 |
| 6 | 馆 | 饣旁与“官”结构明确，右侧不裁切 |

- 六字等高、共用同一基线、视觉字宽一致，字间距误差不超过平均间距的10%。
- 主标题位于画面中轴略偏上，占画面有效宽度约72%—80%，不再追求82%—88%的极限满幅，给标准字形留出呼吸空间。
- 采用整体方正的石刻浮雕字，不再用大量零散小砖拼接每一笔。字面仍有玄武岩矿物纹理、细砂浆边缘和5—8厘米真实厚度。
- 六字必须第一眼可读；材质效果不得破坏字形骨架。

### 副标题

- “敬请期待”置于主标题下方正中，宽度为主标题的30%—36%。
- 四字使用较小、较细但仍清晰的凹刻石字，字面为暖象牙亮芯。
- 不使用逗号、装饰线、英文副题或上映日期。

## 四、背光与投影设计

### 主光

- 一束3000K窄琥珀主光从左上方约35°掠过墙面。
- 凸起主标题向右下方投射清晰但不过硬的实体阴影，阴影长度约为字厚的1.2—1.8倍。
- 右下投影必须服从同一光源方向，不得出现双重反向影子或无来源高光。

### 字背隐藏光

- 主标题背面设置低亮度隐藏光，沿文字右侧与下侧边缘泄出；方向与右下实体投影形成呼应。
- 隐藏光以暖象牙为主体，仅混入约10%—15%的故障青，不把整行字变成霓虹。
- 光晕厚度控制在字高的2%—4%，只照亮字后近邻石面；字面依然由琥珀主光塑形。
- “声”字背光必须完整勾出标准轮廓，不能因高光吞掉中间结构。

### 副标题光效

- “敬请期待”采用更明确的暖象牙内发光，局部故障青只从凹槽边缘渗出。
- 副标题亮度高于墙体但低于主标题最亮石面，形成“主标题震撼、副标题邀请”的层级。

## 五、画面气质

- 旧城近未来现实主义，建筑真实、厚重、潮湿、有使用痕迹。
- 主色为烟木黑、雨夜蓝灰、深夜琥珀、暖象牙、旧黄铜与少量故障青。
- 震撼感来自六字尺度、石材厚度、右下投影、背光轮廓和空间声场，不使用金属Logo、金色艺术字、满墙霓虹、魔法符文、飞砖、爆炸、火花或商业LED墙。
- 画面保留细颗粒和柔和高光滚降，深色区域仍需可读，不压成死黑。

## 六、多角度派生策略

母图未通过前，不生成任何角度图。母图通过后，每张角度图都必须以母图为唯一图像参考，并一次只改变摄影机角度或焦段。

| 资产 | 角度／焦段 | 用途 | 不可变化 |
| --- | --- | --- | --- |
| M01终帧母图 | 正面32mm、轻微低机位 | 最终标题落点与所有资产正史 | 六字字形、排版、墙体、光位、锚钉、裂缝 |
| A01文字微距 | 85mm，靠近“声”字右下边缘 | 动画开场石材、背光和投影细节 | “声”字结构、石材纹理、光源方向 |
| A02左侧斜角 | 50mm，墙面左前方约12° | 展示字厚与向右下投影 | 字距、基线、凸字厚度、背光色 |
| A03低机位全景 | 35mm，低机位仰角约6° | 强化建筑压迫感与标题规模 | 墙体比例、地面、梁柱、标题位置 |
| A04右侧轻斜角 | 50mm，墙面右前方约8° | 为后撤或轻横移提供末端选择 | “馆”字完整、逗号不存在、副标题位置 |

### 连续性调用规则

1. 所有角度图先读取母图审片记录和连续性锚点表。
2. 不允许文本到图像独立重建设计；必须使用母图参考编辑。
3. 一次只改变一个变量：角度、焦段或裁切三者择一。
4. 任何角度图出现错字、字数变化、字距漂移、裂缝跳位、锚钉移动或光向改变，立即废弃，不用于动画。
5. 动画优先使用母图确定性裁切、视差和轻微透视，而不是让视频模型重新生成中文。

## 七、生成与验收顺序

```text
文字与排版提示词锁定
→ 从零生成单一16:9母图
→ 六字计数与逐字人工复核
→ 中文识别辅助检查
→ 光向、投影、背光与材质复核
→ 用户确认母图
→ 建立连续性锚点表
→ 依次生成A01—A04角度图
→ 每张角度图与母图并排审片
→ 进入动画制作
```

### 母图硬闸门

- 主标题逐字为`零1｜号2｜回3｜声4｜酒5｜馆6`，没有第7个大字位。
- 无逗号、句号或其他标点。
- “声”字标准、清晰、无重复、无粘连。
- 六字等高、等距、共用基线，主标题不裁切。
- “敬请期待”准确、清晰，背光克制且方向合理。
- 主光来自左上，实体投影与字背光向右下形成呼应。
- 墙体材质、世界观色卡和电影质感成立。
- 母图输出可打开、尺寸正确、无Logo、水印或额外文字。
