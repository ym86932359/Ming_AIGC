# PV01｜石砖场景化片尾设计

## 一、设计结论

PV01保留60秒故事主体与全部核心内容，只替换60—64秒片尾呈现。原有“黑底后期艺术字”废止，改为一条发生在《零点回声酒馆》世界内部的真实建筑镜头：摄影机从旧城高架与酒馆地下结构交界处的一面烟黑石砖承重墙近处开始，沿墙后撤并轻微仰升，最终发现整面墙的修葺结构本身组成唯一文案“零号回声酒馆，敬请期待”。主标题、中文逗号与副标题均属于墙体结构，不得改写或省略。

文字不是悬浮字幕、投影、霓虹或后期金属字，而是墙体真实结构。震撼感来自建筑尺度、空间透视、掠射光、运镜落点和声场重量，不来自砖块飞舞或Logo爆炸。

## 二、世界观归属

片尾位置设为零号酒馆外墙下方的旧城承重结构层，与高架轨道、旧钟表铺和酒馆内部属于同一建筑群。

| 要素 | 设计 | 与正史的关系 |
| --- | --- | --- |
| 主体石材 | 烟木黑偏冷的旧玄武岩砖，边缘有使用磨损、雨水盐渍与细微缺口 | 继承烟木黑、雨夜蓝灰与真实旧城材质 |
| 修葺字砖 | 比旧墙略新的深灰石砖，突出墙面约6—10厘米；砖缝更规整但仍有人工误差 | 表示酒馆在旧城结构中被重新“缝合”出来 |
| 砂浆 | 暖灰矿物砂浆，局部受2800K琥珀光照亮 | 连接酒馆营业暖光，不引入新色卡 |
| 固定件 | 极少量氧化旧黄铜锚钉与横向加固片 | 呼应门牌、圆钟与吧台黄铜五金 |
| 科技痕迹 | 常态砖缝低于画面1%的故障青残余不主动发光；仅副标题后显时允许内侧砖缝克制透光 | 只提示近未来，不把墙变成赛博广告牌 |

禁止满墙霓虹、纯黑摄影棚、石碑墓园、魔法遗迹、宫殿城墙、砖块自动飞行、火花爆炸和现代商业Logo墙。

## 三、文字建筑结构

### “零号回声酒馆，”

- 六个汉字由真实砖块的排列、凸起高度和砂浆边界共同形成，不再使用平面字体表面。
- 字体骨架保持庄重、方正、略收窄，避免书法飞白、游戏标题尖角和商业招牌圆体。
- 完整字构占画面宽度82%—88%、高度约42%，最后稳定时填满镜头主要视野。
- 每个字内部仍能读出砖的尺寸、接缝、阴影和承重关系；不能像贴在墙上的石纹字体。
- “馆”后的中文全角逗号由较小玄武岩砖构成，位置略低于主标题中轴，但必须与主标题同时清楚可读。
- 主标题最终位于画面中轴略偏上，给下方“敬请期待”留出建筑空间。
- 字砖正面保留少量矿物抛光切面，由左上3000K琥珀掠射光产生暖石银高光；侧面仍为深色玄武岩，不能金属化。

### “敬请期待”

- 使用更小的凹陷砖石镶嵌，位于主标题下方，不采用独立后期字幕。
- 字砖向墙内退2—4厘米，以掠射光和砂浆明度形成可读差异。
- 宽度约占主标题的32%—38%；后显但不抢主标题尺度。
- 后显时由凹砖内部出现暖象牙亮芯，故障青只从局部内侧砖缝透出，光晕厚度控制在字高2%—4%；必须明显可读但不能像霓虹招牌。

## 四、4秒运镜与画面节拍

| 时码 | 摄影机与画面 | 灯光与材质 |
| --- | --- | --- |
| `60.00—60.18` | 60秒整硬切到85mm等效石砖微距，只看见“零”字局部砖缝、湿冷石面和旧砂浆，观众尚不能读出文字 | 近黑环境；仅有墙面潮湿反光与上一镜杯沿余韵 |
| `60.18—61.30` | 摄影机做一次有重量的快速后撤，同时轻微仰升；等效焦段由85mm过渡到约32mm，石砖的凸起视差逐渐暴露主标题结构 | 左上3000K琥珀掠射光沿砖面进入，先照到抛光石材切面、边缘与砂浆，不做全屏光束 |
| `61.30—61.72` | 镜头抵达第一落点，完整“零号回声酒馆，”占据主要画面；后撤速度快速衰减，不急停、不回弹 | 主标题凸砖投下清楚侧影，暖石银高光建立可读性；低频石墙落点同步完成 |
| `61.72—62.45` | 摄影机继续极轻后撤约3%，下方凹陷砖区进入画面，“敬请期待”依次被看见 | 同一琥珀光继续向下，凹砖暖象牙亮芯与极细故障青砖缝透光启动；细砂只从两三处砖缝落下 |
| `62.45—63.50` | 两层内容全部可读；镜头只保留极弱呼吸式稳定，不再新增构图信息 | 远处高架震动使灰尘轻落，砖墙保持实体重量 |
| `63.50—64.00` | 完全锁定终帧，不再推镜、抖动或加新效果 | 光位、亮度与声音同时稳定，留下未解决的期待感 |

运镜只允许一次主运动：近墙后撤＋轻微仰升。禁止无人机旋转、甩镜、穿墙、变焦拉伸、镜头震屏和快速多段推进。

## 五、声音设计

热门电影预告常用“铺垫—上升—落点—尾扣”的结构，但PV01不直接套用通用BRAAAM或爆炸重击。声音必须从既有故事声场自然生长出来。

### 声层与时序

| 时码 | 声音层 | 作用 |
| --- | --- | --- |
| `59.20—60.20` | 原杯沿回声L-cut进入石墙空间，尾音逐渐变成砖石空腔共振 | 让故事场景与片尾建筑发生声学连接 |
| `60.00` | 低于常规预告重击的短促低频落点，主体约48—55Hz；叠加一次闷石接触声 | 支撑硬切，但不惊吓观众 |
| `60.18—61.30` | 反向石砂摩擦、低速空气吸入和远处高架轨道上升层 | 贴合摄影机后撤，建立上映前的期待感 |
| `61.30` | 主标题完整出现时落下一次“石材＋旧黄铜＋低频”的复合重击 | 提供震撼落点；不用爆炸、枪声或通用Logo砰声 |
| `61.72—62.45` | 细砂、短石屑与较亮的玻璃泛音；泛音从记忆杯音色变形而来 | 让“敬请期待”后显，同时保持世界观连续性 |
| `62.45—64.00` | 低电平悬置五度、远轨余震和空间长尾；63.5秒前停止新增声源 | 不完全解决和声，留下即将上映的期待 |

### 混音边界

- 低于120Hz的落点保持中央／单声道兼容；石屑与空间尾音才允许向左右展开。
- 片尾峰值不得超过全片当前真峰值上限，目标`≤ -1.5 dBTP`；整体响度继续维持约`-19至-18 LUFS`，不因“震撼”突然变成广告式暴响。
- 主标题重击相对片尾环境高约6—9dB，但不得遮蔽砖石质感和余韵。
- 不直接复制、采样或仿制具体热门电影、平台Logo或商业音效；只继承现代预告片的结构功能。

## 六、实现架构

- 保留主合成64秒结构，只重做`phase-e-title.html`并将其职责改为`phase-e-masonry`。
- 石墙使用确定性的SVG／CSS砖阵列、矿物噪点、砂浆缝与凸字遮罩构成；准确中文由本地字体路径控制，砖纹填充与侧影让文字成为实体墙体，而不是平面叠字。
- 主标题与副标题分别使用凸砖与凹砖两套深度层；相机运动统一作用于整面墙，灯光与阴影按同一透视移动。
- 新增原创片尾声音`masonry-title-reveal-v03.wav`，由现有杯沿回声、项目环境声与本地生成的石材／低频层合成，不调用外部版权音频。
- v02视频与片尾海报保留；新版输出递增为v03，不覆盖历史资产。

## 八、验收标准

- 60秒硬切后不再出现烟木黑艺术字卡；第一帧即进入世界内石砖墙微距。
- 终帧逐字准确呈现唯一文案“零号回声酒馆，敬请期待”，中文逗号不得缺失。
- 主标题与副标题均由可辨识的实体砖、砂浆、阴影和修葺关系构成；副标题内发光不得破坏凹砖深度。
- 主标题最后占画面宽度82%—88%，完整、不裁切、无乱码，建筑尺度成立。
- 运镜只有一次后撤＋轻微仰升，63.5秒后完全停止。
- 声音能从杯沿回声自然过渡到石墙共振；主落点有震撼感但无广告式突兀暴响。
- v03为1920×1080、24fps、约64秒、48kHz立体声；整片可解码，真峰值与单声道兼容通过。
- Obsidian双链、YAML、H1、片尾命名与版本索引无异常；v02完整保留。
