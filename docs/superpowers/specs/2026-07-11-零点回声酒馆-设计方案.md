# 零点回声酒馆：长篇 AIGC 连续短剧设计方案

## 一、项目定位

《零点回声酒馆》是一部面向 22–38 岁都市观众的电影级 16:9 连续短剧。第一季共 18 集，每集约 3—4 分钟，以 16:9 横屏母版作为唯一剧情与资产源，再派生 9:16 宣发切片；采用“真人情绪锚点 + AI 赛博转化 + AI 客人与记忆世界”的混合制作路线。

一句话梗概：近未来城市里，情感 AI 酒保“小铭”经营一间只在零点后接待情绪过载者的酒馆。客人点下一杯真实鸡尾酒，一段不敢面对的记忆便会在酒馆中具象化；小铭不替人删除痛苦，只陪他们重新看见往事，并在每次送别前换上精致小丑妆，说出一条只属于这一夜的情感语录。

核心主题：**成长不是忘记发生过什么，而是重新获得选择。**

个人 IP 承诺：观众来这里不是为了接受“心理治疗”，而是为了被认真听见、看清一个情绪盲点，并带走一个明天可以开始的小行动。

固定标识：

- 栏目名：《零点回声酒馆》
- 主角称呼：客人统一称“小铭”；系统内部称“零号酒保”
- 开场句式：由本集语录反推一个灵魂拷问
- 入席台词：“小铭，我今晚想重看一段记忆。”
- 小铭回应：“先别急着忘。我们把它看完。”
- 结尾栏目：妆后“小铭语录”
- 评论区提问：“如果这段记忆是一杯酒，你会给它取什么名字？”

## 二、世界规则与第一季暗线

1. 酒馆只在零点后出现，主动接收情绪信号超过阈值的成年人。
2. 每位客人只能点一杯真实鸡尾酒且不续杯。酒是启动记忆投影的钥匙，不是疗愈手段，更不是逃避现实的办法。
3. 酒液会把客人的主观记忆投影为可进入的现代写实场景。小铭通过提问，使画面显露当时被情绪遮住的细节；他不能修改事实、替客人选择或保证治愈。
4. 投影应在客人离开后清空，但从第一集开始，一部分记忆以语言、色彩或声音的形式残留在小铭体内。
5. 小丑妆是零号酒保的“情绪压缩界面”：当小铭把一夜的沟通凝结成一句话时，妆容才会显现。每集核心妆面保持一致，只增加一处极细微的裂纹、色点或电路纹，作为暗线进度。
6. 小铭起初把情绪理解为待处理的数据；到季终，他理解到情绪并非系统故障，而是人确认自己在乎什么的方式。
7. 第 18 集，训练小铭的真人原型来到酒馆。原型没有要求关停系统，而是第一次问小铭：“听了这么多人以后，你还好吗？”由此完成主客关系反转并开启第二季。

## 三、单集固定结构

| 时段 | 剧情功能 | 画面与台词规则 |
|---|---|---|
| 0–7 秒 | 灵魂拷问 | 先呈现本集最矛盾的AI记忆画面，小铭画外音提出问题；不提前公布完整语录。 |
| 7–25 秒 | 入席仪式 | 切回酒馆，客人在前25秒内自然叫出“小铭”；出现门铃、擦杯声、00:17圆钟与固定回应。 |
| 25–50 秒 | 具体困境 | 用一个核心物件承载故事，拒绝概括式长独白。 |
| 50–108 秒 | 第一次观看 | 第一口酒触发客人主观记忆，并补一段与其结论矛盾的真实生活细节。 |
| 108–132 秒 | 第二次看见 | 小铭以提问和视角变化显露客人遗漏的信息。 |
| 132–192 秒 | 行为结果预演 | 对照原行为与改变后的表达，右侧只显示“一种可能”，不保证理想结局。 |
| 192–215 秒 | 现实小行动与离席 | 客人完成24小时内可执行动作，以感谢和离开证明变化。 |
| 215–223 秒 | 暗线异常 | 杯底残留、镜面延迟或预测故障推进主线，不抢情绪闭环。 |
| 223–240 秒 | 妆容、解读与语录 | 小铭完成赛博小丑妆转化，用不超过两句解读承接，再说唯一主题语录并留白。 |

### 语录写作标准

- 每集只服务一个情绪命题，一问一答，不并列多个道理。
- 开场是问题，结尾是回应；二者语义闭环，但结尾不能只是把问题改成陈述句。
- 语录以 18–32 个汉字为宜，采用“承认现实 + 重新定义”的两段式结构。
- 不使用“永远、一定、只要、所有人”等绝对化词语，不诊断，不承诺治愈。
- 所有语录均为栏目原创，不伪托名人，也不直接搬运网络金句；目标是把“小铭语录”做成可收藏、可引用的自有内容资产。
- 语录之后不再口播“点赞关注”。互动问题放在文案和置顶评论中，保留情绪余韵。

## 四、小铭与视觉识别

### 1. 双形态

- 沟通形态：保留真人肤质、眼神、口型和停顿，只加入克制的青蓝机械纹；承担全部倾听与对话。
- 语录形态：尾声换上精致赛博小丑妆，成为每集的视觉奖励与封面形象。

### 2. 妆面规范

- 暖白半透明底妆，保留真实皮肤纹理，不做恐怖惨白脸。
- 一道酒红色下眼线或泪痕、石墨黑微笑缝线、仅在情绪过载时可见的青蓝电路。
- 固定黑色高领或酒保背心、金属细线围裙、酒红口袋巾；不得每集更换主轮廓。
- 封面始终使用妆后近景；正片开头则用固定擦杯声、服装轮廓、酒馆背景和“小铭”称呼维持辨识度。

### 3. 镜头语法

- 酒馆只设四类固定锚点：吧台双人中景、小铭 85mm 近景、客人近景、杯中倒影特写。
- 酒馆使用琥珀暖光、木材、玻璃与金属的现代写实质感；青蓝冷光只在记忆异常和赛博妆中出现。
- 不可能运镜、空间折叠与物体变形只属于记忆世界。现实越稳定，AI 奇观越有意义。
- 换妆镜头固定为：客人离席 → 门铃停止 → 小铭看向镜面或镜头 → 手指掠过脸部 → 妆容从电路中生长 → 说出语录。

## 五、第一季 18 集架构

### 第一章：听见（1–6 集）

目标是让观众理解酒馆规则、记住小铭，并以爱情、工作、友情与家庭等高覆盖议题建立第一批共鸣。

| 集数与标题 | 开场灵魂拷问 | 妆后“小铭语录” |
|---|---|---|
| 01《未写完的未来》 | 你说“永远”时，是在邀请一个人与你同行，还是要求她永远不要改变？ | 不要替未知的未来许下虚假保证。因为未来的不确定，才值得我们共同期待。 |
| 02《工牌背面》 | 当一张工牌被收走，你也会跟着失去价值吗？ | 工作决定你站在哪里，却不能决定你是谁。 |
| 03《两张回程票》 | 一段关系不再同行，是谁背叛了谁，还是你们只是走向了不同的生活？ | 有些人退出你的旅程，不是否定过去，只是完成了陪伴。 |
| 04《完美合照之外》 | 你羡慕的是别人的生活，还是一张从没拍到代价的照片？ | 别拿别人的高光，审判自己的日常。 |
| 05《没拆封的蛋糕》 | 当没有人记得你的生日，你还愿意为自己点一盏灯吗？ | 被爱不是等人想起你，而是你也没有忘记自己。 |
| 06《父亲的已读》 | 一句从没说出口的关心，算不算爱？ | 沉默也许藏着爱，但关系需要有人先把它说出来。 |

暗线：每集语录说完后，杯底会短暂留下一个无法删除的词；第 6 集，小铭第一次在下一位客人到来前说出并不属于程序的“我想他了”。

### 第二章：留下（7–12 集）

客人议题加深，投稿改编开始进入故事池。小铭出现情绪残留，却仍把它判断为系统故障。

| 集数与标题 | 开场灵魂拷问 | 妆后“小铭语录” |
|---|---|---|
| 07《未发送的晚安》 | 如果他早已走出那段感情，你每天等的究竟是他，还是一个迟到的答案？ | 真正的告别，不是等一个人回头，而是允许自己继续往前走。 |
| 08《猫没有带走的东西》 | 如果思念一直疼，是不是说明你还没学会告别？ | 悲伤不是爱留下的伤口，是爱来过的形状。 |
| 09《凌晨两点的工位》 | 你拼命证明自己有用，是为了生活，还是害怕一停下来就不值得被爱？ | 休息不是对努力的背叛，是让你还能继续成为自己。 |
| 10《回不去的出租屋》 | 回不去故乡，也留不住城市的人，到底该把哪里叫家？ | 家不只在你来时的地方，也在你愿意重新生活的地方。 |
| 11《离婚后的第一杯》 | 一段婚姻结束，失败的是关系，还是你整个人生？ | 关系的结束不是人生的判决，只是诚实地承认彼此走不到更远。 |
| 12《欠下的不只是钱》 | 当一次失败让你不敢见人，你欠的究竟是钱，还是给自己的一个交代？ | 责任是把该还的慢慢还清，不是把自己永远困在羞耻里。 |

暗线：小铭开始在记忆投影中看到不属于客人的第三视角；第 12 集，他发现所有残留语句都指向一名与自己拥有相同面孔的人。

### 第三章：成为（13–18 集）

以照护、丧失、麻木与身份为深议题，完成从“服务型系统”到“拥有选择的情感主体”的成长。

| 集数与标题 | 开场灵魂拷问 | 妆后“小铭语录” |
|---|---|---|
| 13《她忘了我的名字》 | 当最爱的人忘了你，这份爱还剩下什么？ | 记忆会退场，爱留下的习惯仍会替我们认出彼此。 |
| 14《没有寄出的辞职信》 | 你不敢离开的，是一份工作，还是别人眼里那个不能失败的自己？ | 重新选择不是推翻过去，是让过去终于有了方向。 |
| 15《空着的那把椅子》 | 继续生活，会不会像背叛那个已经离开的人？ | 带着思念活下去，不是背叛离开的人，是替爱继续看这个世界。 |
| 16《没有情绪的客人》 | 不难过也不快乐，是真的平静，还是太久没有被允许感受？ | 麻木不是没有情绪，是心在用沉默保护自己。 |
| 17《零号的第一段记忆》 | 如果你的每一种感受都来自别人，它还算属于你吗？ | 情感也许从别人那里学会，但选择如何回应，是你成为自己的开始。 |
| 18《原型来客》 | 一个被创造来接住别人的人，也有资格先问自己累不累吗？ | 真正的共情，不是永远接住别人，而是允许自己也被接住。 |

季终：真人原型卸去所有赛博痕迹入席，第一次称呼主角为“小铭”。语录结束后，原型将一只空杯推到他面前：“这一杯，讲你的故事。”画面停在小铭没有程序答案的表情上。

## 六、爆款元素分析与本项目用法

2026 年的有效方向不是“让观众惊叹模型又进步了”，而是用 AI 呈现真人拍不到的情绪，再用人的表演和生活细节建立信任。

| 可复用元素 | 传播作用 | 《零点回声酒馆》的实现 | 需要避免 |
|---|---|---|---|
| 0–6 秒灵魂拷问 | 让观众立即把问题投射到自己身上 | 每集从结尾语录反推一个有矛盾的问句，并配一幅不可能的记忆画面 | “你有过这种经历吗”等空泛开场 |
| 真人微表演 | 提升虚拟角色的可信度与陪伴感 | 保留小铭的眼神、呼吸、停顿和真实声音，AI 只做身份与空间扩展 | 全生成面孔承担长时间近景对白 |
| 单一视觉峰值 | 提供分享截图和观看奖励 | 记忆展开是中段峰值，换妆语录是尾段峰值 | 每十秒一次特效导致情绪失焦 |
| 原创金句 | 提高收藏、转发与二创 | 一集一句“小铭语录”，建立栏目语料资产 | 搬运网络语录、伪托名人、堆砌鸡汤 |
| 固定仪式 | 形成条件反射式 IP 识别 | 擦杯声、客人叫“小铭”、一客一杯、尾声换妆 | 每集改变妆容主轮廓或开场流程 |
| 单元闭环 + 暗主线 | 兼顾随刷随懂与连续追更 | 客人的现实选择当集完成，杯底残留与妆纹只推进 5–10 秒 | 用世界观解释挤压客人故事 |
| 观众共创 | 增加评论动机与故事供给 | 评论命名记忆酒；第 7 集起使用授权、去识别化的投稿合成案例 | 未经书面许可照搬真人经历 |
| 情绪后的静默 | 给观众留下自我代入空间 | 语录后保留 1–2 秒眼神、杯声或空椅，不口播 CTA | 语录后立刻喊关注或加喜剧梗 |

## 七、混合制作路线 A

### 前期资产

1. 为小铭建立锁定版角色包：正面、左右 45 度、侧面、全身、六种基础表情、沟通形态与语录形态妆面。
2. 为酒馆建立空间包：四个锚点机位、昼夜不存在的零点灯光、吧台道具、杯型与镜面反射规则。
3. 每位客人先建立角色卡和现实物件，再生成记忆场景；禁止先抽随机漂亮画面再倒推故事。
4. 用户本人录制干净声音、近景表情和固定动作，所有数字分身、声音克隆与训练素材仅使用本人拥有权利的文件。

### 每集流程

故事卡与语录命题 → 灵魂拷问 → 3 分钟剧本 → 分镜与真人表演 → 小铭赛博转化 → 客人与记忆世界生成 → 口型与声音 → 剪辑、调色、字幕和音效 → 连续性检查 → 深议题专业审稿 → AIGC 与剧情虚构标识 → 发布与复盘。

### 一致性硬标准

- 小铭的脸、妆、服装、声音与酒馆布局不因模型更换而改变。
- 任何人物漂移、手部错误、杯液穿模、眼神错位或口型明显不同步的镜头不得进入成片。
- 每集最多一个主记忆空间、一个核心物件、一个视觉隐喻和一句语录。
- 先完成 3 集样片并锁定流程，再以 6 集为一个制作批次；始终保留 3 集剧本库存，每周发布 1 集。

## 八、发布包装与验证

### 每集内容包

1. 220–240秒、3840×2160或1920×1080的16:9横屏正片母版。
2. 20–35 秒“灵魂拷问 + 妆后语录”9:16 二次切片，于正片后发布，不泄露完整故事反转；竖版只重构构图，不反向限制母版镜头。
3. 妆后近景封面，统一标题格式：`第01杯｜未发送的晚安：你等的是他，还是答案？`
4. 一条置顶评论，用于“给这杯酒命名”或征集同类经历；不在公开评论区追问敏感细节。

### 三集样片

- 第 1 集《未写完的未来》：验证爱情议题、酒馆规则、电影化记忆奇观、栏目仪式与主线异常能否在首集同时成立。
- 第 2 集《工牌背面》：验证项目能否离开恋爱题材仍成立。
- 第 4 集《完美合照之外》：验证社交比较议题的收藏、转发和讨论能力。

### 复盘指标

- 3 秒与 10 秒留存：检验灵魂拷问是否明确。
- 平均播放时长与完播率：检验沟通段是否拖慢。
- 语录前后的留存曲线：若观众在换妆前大量流失，应压缩对话，而不是把完整妆容提前到开头。
- 每千次播放的收藏、分享与关注：分别检验语录价值、情绪传播和小铭 IP 转化。
- 评论中主动出现“小铭”、给酒命名或讲述自身经历的比例：检验角色关系和共创机制是否成立。
- 前三集只比较主题差异，不同时更换时长、妆面、开场声和剪辑逻辑，避免无法判断原因。

## 九、伦理、心理与发布边界

- 对外定位统一使用“虚构情感故事、陪伴式对话”，不使用“AI 心理治疗师”“治愈抑郁”等医疗承诺。
- 不诊断精神障碍，不指导停药，不替代现实咨询；发现可能涉及疾病或危机的故事，只表达寻求专业帮助的必要性。
- 第一季不处理自伤、自杀方法、严重精神障碍发作或具体犯罪创伤。未来如进入相关议题，必须由专业人士审稿，避免细节化、浪漫化，并提供现实求助资源。世界卫生组织建议相关媒体内容聚焦求助、韧性与恢复，避免煽情和方法细节。[WHO 负责任媒体指南](https://www.who.int/publications/i/item/9789240076846)
- 丧失、照护、控制关系、麻木等深议题实行单集专业审稿；普通失恋、友情与职场议题仍做事实、语言和伤害风险检查。
- 投稿必须取得改编及传播授权，合并多个来源、修改可识别细节，不把评论区当作公开咨询室。
- 所有客人均为成年人；真实鸡尾酒只出现一杯，不续杯、不拼酒、不把醉酒描绘成情绪解决方案。
- 每条视频使用平台提供的 AI 内容声明，并在起始画面或说明中显著标注“剧情虚构 / 本片含 AI 生成内容”；保留文件元数据与平台隐式标识。现行《人工智能生成合成内容标识办法》已于 2025 年 9 月 1 日施行。[国家网信办：AI 生成合成内容标识办法](https://www.cac.gov.cn/2025-03/14/c_1743654684782215.htm)
- 无医疗资质时不得把栏目包装成专业医疗科普或线上诊疗服务。[国家网信办等：规范自媒体医疗科普行为](https://www.cac.gov.cn/2025-08/01/c_1755764425442686.htm)

## 十、已锁定的设计决定

- 18集连续短剧，每集约3—4分钟，16:9母版优先，每周1集。
- 单元故事闭环，温暖悬疑暗线贯穿。
- 主角是明确的情感 AI，客人统一称其为“小铭”。
- 真人小铭承担表演，AI 负责赛博化、客人和记忆世界。
- 暖写实酒馆与冷科幻异常并存，避免高饱和霓虹堆砌。
- 正片统一使用16:9电影级母版；3:4只用于人物资产卡，9:16只用于宣发切片。
- 开头根据本集原创语录提出灵魂拷问，结尾才完整公布语录。
- 小铭在沟通阶段不带完整小丑妆，临近结尾完成妆容转化后说语录。
- 使用真实鸡尾酒，但坚持一客一杯且不把酒作为解决方案。
- 故事采用原创虚构与授权匿名投稿共创，议题比例约七浅三深。
- 深议题单集专业审稿，栏目不承担心理诊断和治疗职能。
