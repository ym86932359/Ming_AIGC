## 当前生产状态

- EP01正式剧本：240秒、11个连续场次。
- EP01导演分镜：36镜，时码连续无空档；每镜含表演、摄影、光色、道具、声音、VFX、AIGC要点和负面约束。
- 逐镜连续性：65条角色表演记录、36条色彩调用、86条道具调用。
- 全局资产：1套全局色卡池、4套场景色卡、8个场景、13种材质、10件核心道具与3名当前角色模型。
- 镜头元素：PV01 14/14镜、EP01 36/36镜都已建立人物、配饰、道具、材质、天气、VFX、声音与前后继承链。
- 配饰与表演：3/3角色已有配饰挂点包和动作／口型接口；静态规则完成，动态穿插与5—10秒身份回归视频待测。
- 世界与技术：5张城市空镜已登记，城市分区、标识／UI、天气、摄影镜头、灯闪、VFX和分层交付合同已建立。
- 当前人物视觉：9张新国风赛博琉璃v04人物／妆造／结构／表情资产已生成并通过技术检查，状态统一为`待审`。
- 图片资产：项目内162张图片已完成语义化命名与全量台账登记；人物历史、候选、工程证据和正式母资产均有明确调用边界。
- 历史国漫CG候选：8张早期通用国漫CG图片已改名为`v04-cg02`并隔离在`候选版本/`，不参与当前调用。
- 宣传片：PV01已覆盖为60秒戏谑狞相版、14镜、8段Seedance序列；18张场景/道具母资产继续宣传片调用，KF01当前环境源已切换为带实体招牌“零号酒馆”的v02，KF05/KF07转正片镜中母题参考；1张F01—F04四态卡与6张A/B/C同机位表演板已导演锁定。S01 Take02已在RunningHub完成1920×1080、24fps、约6.06秒带声Seedance输出并通过技术验收，待用户导演终审；S02—S14、正式本人配音和中文口型仍待制作。
- 声音：小铭三态音色圣经和PV01 13句ADR／口型台账已建立；现有预演音频实测为44.1kHz／16bit，正式本人48kHz／24bit母带仍待录制。
- 电影级补全：人物版本原子裁决和50镜元素台账已完成；当前P0阻断为6个场景视觉包、3件道具模型、配饰／动作动态测试、正式声音和20—30秒金样片。

## 制作口径

- 人物资产卡与表情卡目标为3:4竖版，平台实际返回比例必须如实登记；当前9张v04视觉资产实际均为1024×1536（2:3），不拉伸伪装。剧情画面与环境概念图固定16:9，v03及更早版本已完整归档至各角色`历史版本/`。
- 通用人物模板只引用 [[个人创作/ai漫剧_提示词资料库/README|AI漫剧提示词资料库]]，不在项目内复制公共模板。
- 正式分镜必须先调用人物关联包和世界视觉圣经，再写镜头、光影、动作和生成提示词。
- 小铭是用户本人的数字分身；三张本人身份参考、日常总卡和正式全脸小丑妆造卡均已归档，后续只允许通过人物关联包调用。
- 日常赛博酒保与赛博小丑共用同一连续性编号；妆造切换不得改变脸、年龄、发型和天然瞳色。
- 人物、色卡、场景、材质与道具均通过稳定ID调用；分集文件只记录当前版本和状态，不复制全局资产。
- 当前人物画风固定为`ZERO-ECHO-VIS-NEO-CHINESE-CYBER-GLAZE-02`；身份正史高于画风，场景仍保持情感现实主义，不随人物资产板变成满屏霓虹。
