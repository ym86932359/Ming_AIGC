## 历史导演审核结论

- 非废弃视觉资产共21张：5张世界空镜、9张场景关键帧（含KF01 v01历史版与v02当前版）、7张道具模型卡；当前18张继续PV01调用。
- 全部场景与世界空镜采用16:9；全部道具卡采用3:4；除KF01 v02锁定的场内实体招牌“零号酒馆”外，画面内无片名、字幕、Logo或水印。
- 小铭镜头均以v04日常／小丑资产为身份锚点；双形态只发生在妆面、灯光相位、直接硬切、表演和声线状态，不视为两个角色。
- 五张未通过版本已进入`废弃版本/`，不允许作为图生视频参考：KF02椅子数量超标、WK04伪招牌、PROP04铃体比例过大、PROP01与PROP06初版画幅为2:3。
- KF02是室内唯一空间几何母图；KF03—KF08只承担表演、构图、道具或VFX职责。后续图生视频必须同时调用KF02，不能让近景板独立扩写空间。
- PROP07中的环境仅用于展示镜面安装，不继承其中的背景圆钟；所有时间状态必须调用PROP03的00:17正史。

导演审片结论：世界、场景、道具、F01—F04四态卡与A/B/C同机位人物板均达到静态生产要求。下一阶段可进入C01—C08视频测试，但三声线、中文口型和实际运动仍须单独验收。

## 监工硬标准

1. 场景关键帧全部使用16:9电影画幅；除KF01 v02的场内实体招牌“零号酒馆”外，不含片名、字幕、Logo和水印。
2. 旧城与酒馆是生活化近未来，不得变成霓虹夜店、豪华大厅或太空舱。
3. 小铭必须继承同一身份、国漫CG脸部与皮肤、同一服装基础和天然眼眸。
4. 道具模型必须符合正史尺寸、材质、装配和状态，不得只做漂亮概念外形。
5. KF02是所有室内画面的空间母图；后续只改变人物、焦点、杯中效应和灯光相位，不改空间几何。
6. KF06不得连续融脸，只表现可用于硬切合成的一帧灯闪中间态。
7. KF05、KF07不得进入PV01参考列表，只能服务正片镜中母题。
