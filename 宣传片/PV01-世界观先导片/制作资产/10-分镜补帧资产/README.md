1. 只有连续性审核表状态为`DIRECTOR-PASS`的图片才能进入Seedance首尾帧或剪辑图层。
2. 室内帧必须继承KF02空间几何；人物帧必须继承当前锁定A/B/C表演板。
3. IF15、IF16和IF18不得使用生成模型重新绘制人物。
4. IF16只能调用审核通过的IF17，不能调用失败候选。
5. 60.00秒从S14硬切同一石墙微距，62.45秒后才到完整记忆回声渗光v06终点；片尾不加逗号，本目录不生成软过渡帧。
6. `废弃版本/`中的任何图片不得成为后续参考源。
