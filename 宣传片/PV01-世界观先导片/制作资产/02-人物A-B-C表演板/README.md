## 调用规则

1. 所有室内生成同时加载[[个人创作/ai铭仔/宣传片/PV01-世界观先导片/制作资产/01-场景与道具关键帧/场景关键帧/PV01-KF02-零点酒馆主吧台空场-v01.png|KF02空间母图]]；本目录只锁人物、景别、姿态与硬切匹配，不独立扩写酒馆空间。
2. 65mm段使用`A65 ↔ B65-F01/B65-F02`；85mm段使用`A85 ↔ B85-F03/C85-F04`。
3. 常规切换采用直接硬切、最多2帧近黑、一次低强度暖/冷脉冲与最多12px背景失焦；禁止融脸、半脸拼接、镜中变身和双人叠影。
4. S10→S11是全片唯一例外：A65日常与B65/B85小丑层按23帧“明—暗—明—暗—红—近黑—完整小丑”后期合成；红光段只短暂叠现精确重合的小丑脸，不改身份骨相，不生成液化变脸。
5. S07从中性头位开始、句中歪头、最后6帧回中性；S08句中右偏4°、最后6帧回到镜头；S11B最后6帧回正闭口；这些动作端点优先于静态板当前表情峰值。
6. S13—S14只生成一条7秒C85-F04连续源片，57秒无可见切点；F04全段无笑、无歪头、几乎不眨眼。
7. 图内不含字幕、片名、Logo与AIGC标识；文字统一后期叠加。
8. 六张图片是静态身份/表演母板，不代表中文口型、声线与完整动作已经生成。
9. KF05、KF07不得混入PV01参考列表，只服务[[个人创作/ai铭仔/人物资产/小铭/小铭 - 镜中自我对话母题|正片镜中自我对话母题]]。
