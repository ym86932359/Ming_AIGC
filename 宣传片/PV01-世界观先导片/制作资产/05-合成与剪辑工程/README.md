## 工程构成

- 四张2×2彩色分镜板，裁切为Phase A—D共14镜。
- 五个HyperFrames子合成，主时间线64秒、24fps；Phase E为4秒艺术字片尾。
- 13条对白、1条60秒环境声、既有同步音效、独立后景狞笑与片尾显字声。
- 三种角色声线：日常、疯癫小丑、蔑视小丑。
- Noto CJK简体中文字体内嵌，避免跨机器渲染替换字形。
- 项目本地FFmpeg/FFprobe封装、关键帧快照和最终实片核验帧。
- S10→S11的23帧人格夺权、S13—S14的7秒连续F04画面，以及L12/L13重排均已进入v02工程。

本工程用于电影级锁片预演与后续正式视频替换。正式C01—C08镜头接受后，应保持当前时码、转场端点、声音骨架和片尾子合成，以生成视频逐镜替换静态分镜画面。
