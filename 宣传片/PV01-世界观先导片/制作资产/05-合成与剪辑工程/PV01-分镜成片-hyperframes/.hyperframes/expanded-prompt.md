# PV01｜60秒导演分镜成片制作展开

## Title + style block

- 项目：`ZERO-ECHO-PV01`。
- 输出：1920×1080，16:9，24fps，60秒。
- 视觉身份：`frame.md`；背景`#211B19`，文字`#D5CDC1`，现实暖色`#B9793A`，雨夜冷色`#2B3E48`，故障青`#59AAB4`，小丑强调`#77283A`，旧金`#8D7442`。
- 字体：`Songti SC`片名，`Noto Sans JP`对白，`IBM Plex Mono`场记信息。
- 概念角度：观众不是在看一组分镜，而是在走近一扇只为自己出现的门；画面越接近小铭，摄影机越停止，最后被小丑的静止反看。

## Rhythm declaration

`slow-reveal / spatial-breathe / trust-build / FLASH-DEBATE / poison-PUSH / absolute-hold`

能量峰值不是最大闪动，而是S11毒舌落点；真正的模式打破发生在S12→S13——切换不闪，小丑也不再笑。

## Global rules

- 四张2×2分镜板分别裁切为四个阶段；全部保留原始身份与场景，不重新生成。
- 14镜严格覆盖00:00—01:00；S11、S14内部各有一次无缝画面替换。
- 每镜至少三层：BG为原图＋暗角／颗粒；MG为构图；FG为镜号、时码、结构线、声线状态、字幕或进度尺。
- 每镜有且只有一个摄影运动；状态切换镜头优先保持锁定。
- 字幕短句以说话状态分色，每次只显示一组；画外音也标记为`VOICE OVER`。
- 音乐不承担情绪煽动；以雨、列车、门铃、擦杯、玻璃低鸣、4帧抽空和杯沿回声构成声音骨架。

## Phase A｜00:00—00:18｜入口与规则

### S01 00:00—00:03

- Concept：黑暗先有雨声，冷光从积水里“显影”旧城；观众尚未被告知要寻找什么。
- BG：雨巷分镜、冷蓝暗角、细颗粒、低位反射线。
- MG：极慢向门方向推进，画面上方保留黑暗重量。
- FG：`PV01 / S01`、00:00时码、极细冷灰安全框、无字幕。
- Transition：3秒处0.18秒黑色沉切。

### S02 00:03—00:08

- Concept：门缝光比门本身先承认观众；VO第一次说明“记忆可以保存”。
- Motion：缓慢推进到门把手，琥珀边缘亮度轻呼吸一次。
- FG：VO字幕、`ENTRY / 00:17`场记标签、门缝竖线。
- Transition：8秒处0.3秒柔焦交接。

### S03 00:08—00:13

- Concept：空间看起来安全，九席、黑门、圆钟却形成不自然秩序。
- Motion：沿吧台从左向右低速移动；进度尺在底边推进。
- FG：VO字幕、`SC01→SC02`、第七席定位短线。
- Transition：13秒处0.22秒切至物件。

### S04 00:13—00:18

- Concept：杯底一点故障青替代三件物件组接，提示规则已启动。
- Motion：微距慢推，故障青只呼吸一次；字幕收句后留0.4秒玻璃声。
- FG：`INSERT / 90MM`、杯沿测量线、VO字幕。
- Transition：18秒处0.25秒色沉到烟木黑。

## Phase B｜00:18—00:36｜信任与第一次切换

### S05 00:18—00:23

- Concept：小铭通过擦杯、停顿、抬眼让观众相信自己能坐下。
- Motion：4%缓推；人物脸始终清楚。
- FG：日常声线标记`A / DAILY`、对白字幕、镜号时码。

### S06 00:23—00:28

- Concept：放杯动作把服务对象从“酒”改成“记忆”。
- Motion：从杯沿缓慢上移到眼睛；字幕在“不是酒”时锁定。
- Transition：28秒处2帧暗场，所有环境声抽空4帧。

### S07 00:28—00:32

- Concept：同一位置已经是F01小丑；不是变身展示，而是现实界面被替换。
- Motion：镜头锁定；背景仅短暂失焦，人物脸不模糊。
- FG：`B / FERAL / F01`、酒红字幕底、一次短笑波形标记。
- Transition：32秒处1帧暖闪。

### S08 00:32—00:36

- Concept：日常形态恢复得太快，温和台词因此像在掩盖异常。
- Motion：完全静止，仅字幕慢入；视线偏右由原图承担。
- Transition：36秒处单次冷光掠过。

## Phase C｜00:36—00:50｜价值争辩与毒舌峰值

### S09 00:36—00:41

- Concept：F02先快说、短笑、再放慢；观众被点出“故意漏掉”。
- Motion：锁定近景，冷轮廓光脉冲一次；字幕在“故意”上短促强调。
- FG：`B / FERAL / F02`、酒红字幕、一次短笑波形。

### S10 00:41—00:45

- Concept：日常小铭给观众保留体面和退路，画面与声场都松开。
- Motion：无推镜，一次极轻呼吸；字幕使用暖骨象牙。
- Transition：45秒处0.15秒烟木硬切。

### S11 00:45—00:50

- Concept：F03第一次越过吧台的安全距离；“怯懦”落点后笑意突然死亡。
- Motion：前2.2秒缓慢推近，随后切到更紧画面并完全静止。
- FG：`B / FERAL / F03`、酒红字幕、`65→85MM`、距离刻度。
- Transition：50秒处无闪直接进入日常近景。

## Phase D｜00:50—01:00｜分工与蔑视邀请

### S12 00:50—00:53

- Concept：日常小铭不反驳，只说明自己不替人遗忘；这是最后一次保护观众。
- Motion：完全锁定；声场趋近真空。
- FG：`A / DAILY`、暖骨象牙字幕、眼神水平线。
- Transition：53秒处零帧视觉特效的像素级硬切。

### S13 00:53—00:57

- Concept：F04不笑、不歪头、不眨眼；疯癫消失后，蔑视显得更接近真相。
- Motion：所有环境装饰停止呼吸；字幕以更慢节奏出现。
- FG：`C / CONTEMPT / F04`、酒红细线、片名安全区。
- Transition：57秒处普通硬切，不再追加闪动。

### S14 00:57—01:00

- Concept：小丑不再替观众回答；他说完“敢看吗”后继续凝视，片名才出现。
- Motion：前2.2秒极轻推近，最后0.8秒切至更紧静止帧，所有运动停止。
- FG：对白字幕在59.0秒前退出；59.1秒片名`零点回声酒馆`慢显，副标`导演分镜预演`与`ZERO ECHO / PV01`出现。
- Transition：60秒色沉到烟木黑。

## Recurring motifs

- 圆钟／时码：每镜左上角以`IBM Plex Mono`记录，片尾与背景圆钟呼应。
- 故障青：只在杯、切换边缘和细进度标记出现，画面占比不超过5%。
- 酒红：只属于小丑字幕与极细低位线，不用于血腥联想。
- 一条下方进度尺贯穿全片，在状态切换时短暂停住。

## Negative prompt

- 禁止网页式居中卡片、圆角UI、渐变字、满屏HUD、网红字幕、每镜相同缩放。
- 禁止镜中小丑、连续融脸、频闪、眼睛发光、额外角色、字幕压脸。
- 禁止把静态分镜强装成完成动画；保留“导演分镜预演”身份。
