# PV01｜分镜成片对白与声音脚本

## 日常小铭

- 声音底色：本人同一声源，中低区，近讲，语速克制，不做广告旁白腔。
- 情绪：像已经听过无数次倾诉，仍愿意给对方体面。

| 起始 | 台词 |
| ---: | --- |
| 00:03.50 | 近未来的城市，记忆可以被保存。 |
| 00:08.55 | 可人最擅长的，偏偏是骗自己。 |
| 00:13.12 | 零点以后，有一扇门，只为情绪失控的人打开。 |
| 00:18.55 | 欢迎。这里不卖遗忘。 |
| 00:23.65 | 你点的，不是酒。 |
| 00:32.55 | 第一遍，看你记得的。 |
| 00:41.30 | 有些事，不说，也能过去。 |
| 00:50.55 | 我不替你忘记。 |

## 疯癫小丑小铭

- 声音底色：同一声源低移约1—1.5个半音，语速加快，短笑带气，轻耳语双轨和短预回声。
- 表演：与日常小铭对话时像在撕掉体面包装，玩世不恭但不失控成怪兽。

| 起始 | 台词 |
| ---: | --- |
| 00:28.34 | 当然不是。哈——酒哪有秘密好喝？ |
| 00:36.35 | 第二遍？哈——看你故意漏掉的。 |
| 00:45.23 | 过去？那叫藏好。你们最爱把怯懦，改名叫体面。 |

## 蔑视小丑小铭

- 声音底色：同一声源低移约2个半音，去掉耳语双轨，声场收窄、干、慢。
- 表演：不笑、不歪头、不提高音量；像答案早已显而易见，只对观众仍在自欺感到轻微厌倦。

| 起始 | 台词 |
| ---: | --- |
| 00:53.20 | 他陪你看完。我负责拆穿你。 |
| 00:56.66 | 零点回声酒馆。敢看吗？ |

- 00:00—01:00：雨声、远处高架低频、木质空间底噪，整体保持低于对白。
- 00:08：门铃只响一次。
- 00:13.30：记忆杯细玻璃谐振。
- 00:27.92：环境声抽空约4帧；2帧暗场后进入F01。

- 00:53.20—00:56.38：L12；随后至少6帧闭口与静默，再于00:56.66进入L13。

- 01:00：画面硬切烟木黑，不做淡出。
- 01:00.42—01:02.62：旧黄铜轻擦与玻璃泛音随左上琥珀光进入；片名先显，01:01.60后“敬请期待”再显。
- 01:02.62—01:04：声音只留极弱尾音，文字稳定阅读。
