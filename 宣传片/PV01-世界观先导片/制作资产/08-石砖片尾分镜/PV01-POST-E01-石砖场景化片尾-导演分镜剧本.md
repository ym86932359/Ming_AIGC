# PV01 POST-E01｜石砖场景化片尾导演分镜剧本

## 一、段落任务

60秒故事结束时，小丑小铭的凝视不溶解、不渐隐，直接硬切到世界内部的石砖承重墙。观众先看到无法辨认的局部修补痕迹，再通过一次有重量的后撤运镜发现：整面旧墙被重新修葺成唯一完整文案“零号回声酒馆，敬请期待”。作品名不再是一张片尾卡，而成为这座城市真实存在的一部分。

## 二、统一空间合同

| 项目 | 锁定内容 |
| --- | --- |
| 场景 | 零号酒馆与旧城高架共用的地下承重结构层；同一面烟黑玄武岩砖墙 |
| 唯一文案 | “零号回声酒馆，敬请期待”；中文全角逗号是文案的一部分，不得删除或替换 |
| 主标题 | “零号回声酒馆，”，深灰玄武岩凸砖修葺，突出墙面6—10厘米；正面有暖石银矿物高光 |
| 副标题 | “敬请期待”，较小凹砖镶嵌，内退墙面2—4厘米；暖象牙亮芯＋极细故障青内侧砖缝透光 |
| 砂浆 | 暖灰矿物砂浆，人工修补误差可见，不能像机器打印 |
| 固定件 | 极少量氧化旧黄铜锚钉和短加固片，不形成装饰边框 |
| 光线 | 左上3000K窄掠射光；基础环境为雨夜蓝灰与烟木黑；主标题靠石材反射与局部对比提亮 |
| 科技 | 仅“敬请期待”内侧砖缝允许故障青主动透光，光晕厚度为字高2%—4%；禁止商业霓虹 |
| 空气 | 潮湿石墙、少量盐渍、两三处细砂落下；无浓雾、火花和粒子雨 |
| 摄影 | 一次主运动：重型后撤＋轻微仰升；无旋转、穿墙、震屏和甩镜 |

## 三、逐镜拆解

### E01-A｜60.00—60.18｜硬切石墙微距

| 字段 | 导演合同 |
| --- | --- |
| 时长 | 0.18秒，约4帧 |
| 景别／焦段 | 极端特写，85mm等效；对焦距离约0.55米 |
| 机位 | 摄影机几乎贴墙，位于主标题左上区域，略低于“零”字上部凸砖 |
| 构图 | 画面只见“零”字一处横竖砖缝、湿玄武岩表皮、暖灰砂浆与一道浅侧影；不能读出完整汉字 |
| 运动 | 前4帧近乎静止，只允许镜头稳定器残余小于0.2%的呼吸 |
| 灯光 | 近黑；上一镜琥珀已消失，只剩潮湿石面极弱冷反光 |
| 声音 | S14杯沿回声L-cut进入；60秒整叠一次克制的48—55Hz低频落点和闷石接触声 |
| 转场 | 从S14凝视硬切，不溶解、不闪白、不插纯黑帧 |
| 结束状态 | 砖缝方向和侧影建立，摄影机准备沿墙法线后撤 |
| 禁止 | 完整文字、艺术字、石碑、墓墙、震屏、爆炸和飞砖 |

### E01-B｜60.18—61.30｜后撤发现修葺结构

| 字段 | 导演合同 |
| --- | --- |
| 时长 | 1.12秒，约27帧 |
| 景别／焦段 | 从85mm等效过渡到约44mm视野；不是镜头变焦，而是摄影机真实后撤造成透视变化 |
| 机位／轨迹 | 沿墙法线后撤约4.5米，同时仰升约0.65米；水平偏移小于画宽2% |
| 构图 | 同一面墙逐渐出现多个凸砖笔画，观众可以猜到是汉字，但画面边缘仍裁掉最后两个字和副标题 |
| 运动节奏 | 60.18起加速，60.76达到最高速度，61.05后开始减速；有重量但不使用速度拉伸 |
| 灯光 | 左上2800K掠射光比摄影机晚2帧进入，沿砖面斜向移动；只照亮凸砖边缘、砂浆和潮湿高光 |
| 材质动作 | 两处细砂因高架低频震动从砖缝落下，持续不超过12帧 |
| 声音 | 反向石砂摩擦、低速空气吸入和远轨上升层同步后撤；没有电子Whoosh |
| 结束状态 | 主标题已有约75%进入画面，字墙身份即将被完整揭示 |
| 禁止 | 砖块自行拼字、镜头旋转、无人机运动、运动模糊遮字和额外灯源 |

### E01-C｜61.30—61.72｜主标题建筑落点

| 字段 | 导演合同 |
| --- | --- |
| 时长 | 0.42秒，约10帧 |
| 景别／焦段 | 32mm等效建筑宽景；机位略低于主标题中轴，保留墙体压迫感 |
| 构图 | 完整“零号回声酒馆，”进入画面，宽度占画面82%—88%、高度约42%；六字与中文逗号完整、端正、不裁切 |
| 运动 | 后撤速度在10帧内按`power3.out`式重型减速，抵达第一落点；无回弹 |
| 灯光 | 3000K琥珀掠射光照清凸砖抛光正面与右下侧影，形成暖石银矿物高光；旧黄铜锚钉只产生点状窄高光 |
| 声音 | 61.30同步一次“玄武岩闷击＋旧黄铜短击＋中央低频”的复合落点，强于环境6—9dB但不超过全片峰值 |
| 结束状态 | 主标题完全可读；下方副标题区域仍暗，只见模糊凹砖轮廓 |
| 禁止 | 主标题自发光、金属Logo、边框、字幕条、满屏粉尘和预告片通用BRAAAM |

| --- | --- |

| 运动 | 62.45后只保留低于1%的稳定呼吸；63.50起完全锁定，最后12帧无任何新运动 |

| 材质动作 | 两三粒砂砾落下后停止；墙面、锚钉、砂浆与砖纹持续一致 |

| 终帧 | 唯一文案“零号回声酒馆，敬请期待”全部清晰，中文逗号准确，主标题仍占画面82%—88%，灯光和相机完全稳定 |
| 禁止 | 错字、缺逗号、新文字、英文、拼音、水印、AIGC标签、霓虹招牌、继续推镜、光束扫出墙外和结束前淡黑 |

## 五、GPT Image与动态制作边界

- GPT Image四格板使用2×2布局，同时呈现E01-A/B/C/D；它用于审查同墙连续性和运镜端点。
- 四格板不得出现镜号、时码、英文技术标注或额外字幕，减少模型文字错误；技术信息只保存在本文件。
- 视觉板通过后，另生成一张只有E01-D终帧的4K母图。HyperFrames从母图做确定性裁切、视差、灯光和砂砾动画，不让视频模型重生中文字。
- 如果四格板中文字错误、中文逗号缺失或副标题不够醒目，优先基于同一图局部编辑；不得用随机新墙替换导致材质与构图漂移。
