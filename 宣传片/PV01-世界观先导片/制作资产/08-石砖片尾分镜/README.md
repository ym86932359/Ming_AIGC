## 单一来源规则

1. 四格板只决定同一面墙的构图、材质、光位和运镜端点，不直接作为发布画面。
2. 正式动态片尾只能调用通过审片的终帧石墙母图，再以同图裁切完成近景到全景的后撤运镜。
3. 终帧唯一文案严格锁定为“零号回声酒馆，敬请期待”，中文逗号不得删除、替换或移位；主标题与副标题均为实体砖石结构。
4. 主标题只靠更亮的石材切面与琥珀掠射光显形；“敬请期待”允许暖象牙字面亮芯与极细故障青砖缝透光，但禁止变成平面字幕、商业霓虹、LED字或投影。
5. 当前v01由平台原始生成文件直接复制入库，SHA-256与生成源一致；不得再用会话截图替换它。
6. 新版递增版本号，不覆盖v01视觉板或历史v02锁片视频。
7. 主标题必须只有六个大字位：`零｜号｜回｜声｜酒｜馆`；中文逗号是小型标点，不计入大字位。任何重复“声”或第七个大字位均判定为失败。
8. 六个大字必须按视觉等距排列；任何删除字后留下的异常空槽均判定为失败。
