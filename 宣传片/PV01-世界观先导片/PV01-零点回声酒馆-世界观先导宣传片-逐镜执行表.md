# PV01｜零点回声酒馆世界观先导宣传片逐镜执行表

## 制作原则

- 一镜一个主要任务；身份、口型和妆造脆弱时使用锁定机位。
- 宣传片不使用镜中小丑、镜像对话或镜子完成变身。
- 日常、疯癫小丑、蔑视小丑分别生成A/B/C表演板，经灯闪与硬切合成。
- 生成画面不承担片名、字幕和AI标识文字，全部后期叠加。
- 室内镜头必须同时调用KF02空间母图；KF05、KF07退出PV01当前调用。
- 十三组镜间转场与60—64秒片尾艺术字，以[[个人创作/ai铭仔/宣传片/PV01-世界观先导片/PV01-零点回声酒馆-电影级转场与声画锁片表|电影级转场与声画锁片表]]为唯一正史。

### 场景底板

1. KF02无人物酒馆主景，全部营业灯开启。
2. KF02同机位暗化底板：琥珀降1档、冷轮廓升高，不改空间。
3. 第七号位记忆杯微距干净板。
4. S06—S13共用的65mm/85mm吧台正机位干净板。

### 日常小铭A板

1. S05擦杯与抬眼完整动作。
2. S06使用65mm放杯与“你点的不是酒”，末帧必须匹配A65。
3. S08、S10、S12锁定机位短句板。
4. 无台词中性凝视10秒，便于匹配切点。

### 疯癫小丑B板

1. S07／PV-CLOWN-F01：从中性头位开始，轻笑、歪头、狞笑试探，最后6帧回中性。
2. S09／PV-CLOWN-F02：快—停—快与一次短“哈”。
3. S11／PV-CLOWN-F03：贴近半步、毒舌落点回正、突然停笑。
4. 每条先录干净短句口型，不在同一条完成多次切换。

### 蔑视小丑C板

1. S13／PV-CLOWN-F04：无笑、无歪头、无多余眨眼。
2. S13与S14优先生成一条7秒连续表演；57秒只作为语义节拍，结尾保留0.8秒静止尾帧。
3. 整条不加入闪动、镜面或夸张耳语；如必须拆段，只能用S13接受末帧继续。

## 后期安全阈值

- 任何闪光不得形成连续高频频闪。
- 人脸、妆纹、眼睛和嘴部锐度高于背景故障。
- 五次状态切换均遵守A/B/C身份硬锁；只有S10→S11允许短暂酒红显影叠层，仍禁止连续变脸。
- 字幕、片名、剧情虚构／含AI生成内容及发布信息全部后期叠加。
- 声音需人工检查齿音、爆破音、短笑响度、三声线相位和中文口型。
- 毒舌只揭露主动掩饰的借口，不攻击外貌、身份、疾病、创伤或不可控弱点。
