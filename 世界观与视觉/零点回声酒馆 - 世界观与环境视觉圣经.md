# 零点回声酒馆 - 世界观与环境视觉圣经

## 一、世界观基调

故事发生在未标明具体年份的近未来，技术比现实前进约十至十五年。人仍然租房、通勤、失恋、失业、照护家人；城市不会因为科幻而失去生活痕迹。算法能够预测行为、保存记忆、模拟关系，却依然不能替人决定如何面对变化。

核心视觉命题：**熟悉世界里出现一处无法被完全解释的偏差。**

科幻信息不靠满屏悬浮界面，而通过四种现象显露：

1. 鸡尾酒中的青蓝雾能承载记忆。
2. 酒馆镜面有半秒至两秒的时间延迟。
3. 人工智能皮肤下只有极细的青蓝电路，不铺满全脸。
4. 城市基础设施比今天更智能，但仍依附于旧街区、轨道、广告灯箱和雨水管线。

## 二、城市环境

### 城市层级

- **上层新区**：玻璃、再生混凝土、自动轨道、无人配送与精确规划；色彩冷、空间宽、人少。
- **旧城区**：高架轨道、旧商住楼、雨巷、便利店、维修店与深夜餐馆；材料有磨损，生活声更密。
- **记忆层**：不是另一座城市，而是现实空间在情绪影响下重新排列；门、走廊、车站和房间反复出现。

### 零点入口

酒馆位于旧城区高架轨道下的一条支巷。白天是一间封闭的旧钟表维修铺；零点后，黄铜门牌才显出“ZERO ECHO”。普通路人只会看见熄灯卷帘门，情绪信号超过阈值的成年人会看见一扇琥珀色木门。

入口固定物件：

- 被雨水氧化的黄铜门牌。
- 停在 `00:17` 的小型圆钟。
- 只有来客接近才亮起的暖白门灯。
- 路面积水映出的青蓝轨道光。
- 不出现大字号霓虹招牌、品牌Logo或拥挤赛博广告。

## 三、酒馆主场景

### 空间结构

- 约18米长、4.8米宽的狭长空间，一点透视纵深明显。
- 左侧为九个座位的深色胡桃木吧台；第七号座位为主要客席，零号空椅永远不移动。
- 右侧是烟熏玻璃酒柜和窄通道，尽头是一面略有旧化的长镜。
- 吧台后方有一扇无把手黑门，只在主线异常时出现极细青光。
- 天花保留旧建筑梁柱和黑化钢轨构件，科技设备藏在木材、玻璃和镜面内。

### 材质

| 对象 | 材质要求 | 禁止 |
| --- | --- | --- |
| 吧台 | 深色胡桃木、细划痕、局部水渍、哑光保护油 | 塑料木纹、无瑕奢侈品柜台 |
| 金属 | 旧黄铜、拉丝黑钛、少量氧化边缘 | 镜面铬、网游发光金属 |
| 玻璃 | 有厚度、冷凝水、真实折射、烟熏反射 | 透明塑料、过度彩虹色散 |
| 墙面 | 暖灰矿物灰泥、局部修补痕、吸音织物 | 太空舱白墙、大面积LED墙 |
| 地面 | 深灰磨石、雨水脚印、轻微反光 | 完美镜面地砖 |

### 固定叙事道具

- 黄铜门铃：来客与离席各响一次。
- 黑钛调酒匙：小铭的核心手部道具。
- “未定式”杯：透明酒液中有不沉淀的青蓝雾。
- 长镜：主线异常时镜像慢半秒。
- 零号空椅：暗示小铭终会坐到客人的位置。
- 圆钟 `00:17`：酒馆时间并非正常连续。

## 四、全局色卡

唯一正式色值、RGB/HSL、比例范围、材质职责和禁用位置保存在[[个人创作/ai铭仔/资产库/色彩资产/零点回声酒馆 - 全局色卡池|全局色卡池]]。本圣经只保留美术方向：琥珀现实、烟木暗部、雨夜蓝灰与不超过5%的故障青；禁止高饱和洋红、荧光蓝、商业青橙、糖果紫、死黑和无来源金色能量。

## 五、灯光合同

### 基础营业状态

| 光源 | 参数 | 功能 |
| --- | --- | --- |
| 吧台实景灯 | 2700–2900K，柔和下照，亮度100% | 建立安全与亲密感 |
| 人物主光 | 左前35°、高于眼睛20°、4300K、中等柔光 | 保护真实肤色、眉眼和停顿表演 |
| 辅光 | 右前20°、5000K、主辅光比4:1 | 保留暗部眼窝与黑色服装纹理 |
| 雨窗轮廓光 | 左后45°、6800–7200K、35% | 分离肩线与发丝，不染蓝皮肤 |
| 眼神光 | 镜头上方小型柔光点、低强度 | 保留角膜湿润，不做LED眼 |

基础曝光：`-0.3EV`；肤色 `58–64 IRE`；玻璃与黄铜高光不超过 `88 IRE`；黑位保留在 `4–7 IRE`。

### 记忆启动状态

- 实景灯整体下降约1.5档，人物主光只下降0.5档，保证脸不消失。
- 故障青只来自杯中雾、镜面边缘和极少量皮下电路，画面占比不超过5%。
- 空间阴影向蓝灰偏移，暖木色仍存在，不能全场变成青蓝科幻棚。

### 妆后语录状态

- 背景实景灯依次熄灭，只保留3800–4200K的正侧柔光和7200K克制轮廓光。
- 小丑底妆仍保留皮肤纹理；酒红泪痕不发光，青蓝电路只沿边缘亮起。
- 语录结束后不补高燃逆光或炫光，以静默和空白日历收束。

## 六、电影画面基准

- 输出：3840×2160或等比例16:9，无额外黑边。
- 建立空间：24–28mm，机位高1.45–1.60米，保持真实透视。
- 双人对话：40–50mm，避免过度长焦把狭长空间压平。
- 情绪近景：75–85mm，浅景深但必须保留一处环境关系。
- 道具与眼神：90–100mm微距，控制景深，不把物件拍成广告。
- 镜头运动：一镜一个主运动；固定、缓慢横移、轻微推近、低速跟拍优先。
- 影像质感：克制细颗粒、柔和高光滚降、极轻晕光、真实皮肤和材质；不使用过重镜头炫光和数码锐化。

## 七、记忆世界的变化规则

记忆世界必须从客人的主观理解出发，第二次观看才暴露被忽略的事实。视觉变化遵循：

1. 第一次观看：构图精确、空间稳定、色彩受客人叙述控制。
2. 小铭提问后：环境中的重复、缺失和不合理关系逐渐可见。
3. 第二次观看：镜头改变立场而不是简单换滤镜；例如从跟随林野改为穿过姜禾所在的房门。
4. 认知变化后：空间不爆炸，结构松开、门打开、雾散开或环境声恢复。

## 八、环境禁区

- 满墙霓虹、繁杂悬浮HUD、赛博朋克夜店、全息广告海洋。
- 每个画面都下雨、冒烟、逆光和浅景深。
- 无生活痕迹的豪华酒吧、纯黑背景棚拍、塑料CG材质。
- 科幻技术抢过人物情绪，或为“高级感”加入无功能装饰。
- 直接模仿具体电影IP、现成人物、品牌酒标和真实商业Logo。

## 九、年轻化仿真人人物合同

- 正式人物采用约`70%真人电影质感＋30%可解释近未来技术`，科技必须落在真实面料、磁吸闭合、电子墨水织线、感应节点和功能道具上。
- 年轻化通过软组织紧致度、轻微减少疲态、发型与服装轮廓完成；禁止缩短中庭、放大眼睛、抹平鼻翼下颌或生成少年偶像脸。
- 小铭保持本人可辨识身份并克制年轻约5岁；林野正式年龄26岁；姜禾正式年龄24岁。
- 科技发光面积不超过人物画面3%，必须接受环境光、产生真实反射、遮挡、受力和阴影。
- 人物皮肤保留毛孔、绒毛、唇纹、泪膜与自然不对称；禁止游戏CG脸、网红磨皮和塑料皮肤。

## 十、人物赛博琉璃与场景边界

- 人物资产统一服从[[个人创作/ai铭仔/世界观与视觉/零点回声酒馆 - 新国风赛博琉璃人物视觉风格规范|`ZERO-ECHO-VIS-NEO-CHINESE-CYBER-GLAZE-02`]]：仿真人身份与人体结构50%、新国风造型结构20%、动漫化3D CG材质20%、电影灯光与视效10%。
- 赛博琉璃是人物服装与道具的局部材质，不是整个世界的表面。酒馆、街区、记忆空间继续使用真实木材、旧金属、玻璃、雨水和生活磨损。
- 青紫炫光只允许描出人物发丝、琉璃边缘和外轮廓；场景仍由琥珀实景灯、雨夜蓝灰和不超过5%的故障青主导。
- 对话近景必须优先保护肤色、天然虹膜、眼白和微表情；不得为了匹配人物资产板，把酒馆改成赛博夜店或让青紫光覆盖整张脸。
- 三名角色的琉璃语言必须可区分：小铭为黑曜青玉与东方非对称秩序；林野为冷烟矩形预测晶体；姜禾为珠光开放弧面与轻透明层。
