## 制作规则

- 每集建立独立制作包，正式剧本、导演分镜、表演连续性、色卡与道具调用分开维护。
- 跨集资产只从[[个人创作/ai铭仔/资产库/README|全局生产资产库]]和[[个人创作/ai铭仔/人物资产/README|人物资产中心]]调用。
- 正片使用16:9电影级母版；人物资产使用3:4竖版。当前v04新国风赛博琉璃人物与表情资产为唯一正式调用源，v03及更早版本只在历史目录追溯。
- 每镜必须绑定场景色卡、角色表演状态和道具状态；不得只写“电影感”或“悲伤”。
